# 现代美国版三国 · 懂王实验版

> 分支：experiment/懂王
> 董卓 = 唐纳德·特朗普

## 分支名含义

「懂王」三重含义：
1. **董**——董卓的董
2. **王**——King（支持者称他为 God Emperor / The King）
3. **懂王**——中文互联网「懂王」的谐音

## 本分支特色

- 董卓直接映射为唐纳德·特朗普——不换皮、不隐喻，他本身就是三国式人物
- Brother Wang 是MIT金融博士、华尔街量化天才，从未出现在官方顾问名单上的影子军师
- 入京方式是选举夺权而非军事入侵——建制派「招安」特朗普，以为能控制他，结果被反噬
- 冲击国会山 = 火烧洛阳，海湖庄园 = 郿坞
- 副总统（彭斯式人物）= 吕布，关键时刻背叛

## 核心人物

董卓阵营独有角色：
- **唐纳德·特朗普**（董卓）：纽约地产商/真人秀明星/总统，被精英视为野蛮人
- **Brother Wang**（李儒/许临）：MIT金融博士→华尔街量化天才→影子军师，金融兵法，围棋高手
- **李儒**：特朗普女婿（库什纳式人物），管理商业帝国和竞选资金

共通角色见 `世界观设定-共通.md`：曹操、孙坚、刘备、司马懿、燎原火/赵云、小孟、王允、王蕊、丁原、吕布、残兵等。
