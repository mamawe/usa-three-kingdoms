# 现代美国版三国

以《三国演义》精神为骨架，现代美国为舞台，参考《火凤燎原》叙事手法创作的政治军事史诗世界观。

## 故事梗概

> 「天下大势，分久必合，合久必分。」

2026年，加密货币「自由币」崩盘引发全美金融暴动，三千万人毕生积蓄化为乌有。联邦政府陷入宪法危机，各州州长趁机扩充军队，一个来自边缘地带的「野蛮人」率军进入华盛顿，废黜总统，自任独裁者。各州联合组建「宪法恢复联盟」讨伐入侵者，但联盟内部各怀鬼胎。在混乱中，三个出身卑微的人正在悄悄崛起，而在华尔街，一个28岁的对冲基金经理正在用算法和暗杀网络编织一张看不见的网。

## 核心共通人物

- **曹操**：宾州国民警卫队中校，父亲是五角大楼腐败采购官（「史密斯专员」），军事天才，速度兵法
- **孙坚**：海岸警卫队少校，墨西哥湾反走私猛将，联盟先锋，将缴获「联邦核武密码箱」（传国玉玺）
- **刘备**：基层社工，五金店倒闭后帮助无家可归退伍军人，人心兵法
- **司马懿**：28岁华尔街对冲基金经理，残兵组织幕后资助者，信息兵法
- **燎原火/赵云**：残兵首领，天生无痛觉，以修车铺老板「火哥」身份潜伏
- **小孟（貂蝉）**：残兵渗透专家，跨性别女性，与王允之女王蕊相恋
- **王允**：共和党党鞭，主管国会秩序与议会礼仪，表面合作暗中策划政变
- **吕布**：前海豹六队精英突击手，丁原义子→投奔董卓→后背叛董卓
- **丁原**：北方军区司令，三星中将，吕布的义父

## 叙事特色

- **计中计**：每个关键事件至少三层反转，暗线不标注，读者第二遍才拍桌子
- **限知视角**：叙述者不解释、不提示、不点破
- **火凤式节奏**：以极慢节奏讲极早期故事，董卓篇即独立完整的三幕剧
- **小人物视角**：咖啡店老板娘、联储运维工程师、卡车司机——在关键场景穿插平民视角
- **多分支实验**：两个 experiment 分支分别探索不同的「董卓入京」可能性
