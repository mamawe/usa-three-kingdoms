# 现代美国版三国

以《三国演义》精神为骨架，现代美国为舞台，参考《火凤燎原》叙事手法创作的政治军事史诗世界观。

## 故事梗概

> 「天下大势，分久必合，合久必分。」

2026年，加密货币「自由币」崩盘引发全美金融暴动，三千万人毕生积蓄化为乌有。联邦政府陷入宪法危机——总统被幕僚长架空，副总统拒绝充当傀儡，白宫陷入「无人驾驶」。司法部长在绝望中秘密联络了一支DEA在南美培养的反毒武装——由哥伦比亚裔指挥官塞巴斯蒂安·莫拉莱斯领导的「南方军团」。

莫拉莱斯率军从墨西哥湾登陆，沿I-55/I-40/I-81走廊直插华盛顿，成为第一个占领白宫的外来势力。他废黜总统，指定傀儡「少帝」，自任「联邦紧急状态特别执政官」。他的影子参谋长 Brother Wang——一个北大毕业、在拉美地下世界被称为「王哥」的华裔金融天才——在联邦储备系统中植入了看不见的后门。

各州州长纷纷扩军自立。31个州的代表在丹佛成立「宪法恢复联盟」，推举加州州长为盟主，宣誓讨伐莫拉莱斯。但联盟内部各怀鬼胎。在混乱中，三个出身卑微的人正在悄悄崛起：宾州国民警卫队中校曹操、路易斯安那海岸警卫队少校孙坚、退伍军人事务部社工刘备。而在华尔街，28岁的对冲基金经理司马懿正在用算法和暗杀网络编织一张看不见的网——他的棋子已经安插在每一个关键位置。

华盛顿的夜晚很安静。但不会安静太久了。

## 核心人物

- **曹操**：宾州国民警卫队中校，父亲是五角大楼腐败采购官（「史密斯专员」），军事天才，速度兵法——「在敌人制定计划之前已经赢了」
- **孙坚**：海岸警卫队少校，墨西哥湾反走私猛将，联盟先锋，将缴获「联邦核武密码箱」（传国玉玺）
- **刘备**：基层社工，五金店倒闭后帮助无家可归退伍军人，人心兵法——「不需要攻城，只需要让人愿意打开城门」
- **莫拉莱斯（董卓）**：DEA南美武装首领，丛林兵法——「你们在华盛顿建了监狱和坟墓，我至少知道怎么让穷人活下去」
- **Brother Wang**：北大→拉美卡特尔影子CFO→董卓大脑，金融兵法——「不需要杀人，只需要让对方的钱变成废纸」
- **司马懿**：28岁华尔街对冲基金经理，残兵组织幕后资助者，信息兵法——「谁控制了信息谁就控制了现实」
- **燎原火/赵云**：残兵首领，天生无痛觉，以修车铺老板「火哥」身份潜伏在曹操腹地
- **小孟（貂蝉）**：残兵渗透专家，跨性别女性，以「Celeste」身份渗透华盛顿社交圈，与王允之女王蕊相恋

## 叙事特色

- **计中计**：每个关键事件至少三层反转，暗线不标注，读者第二遍才拍桌子
- **限知视角**：叙述者不解释、不提示、不点破
- **火凤式节奏**：以极慢节奏讲极早期故事，董卓篇即独立完整的三幕剧
- **小人物视角**：咖啡店老板娘、联储运维工程师、卡车司机——在关键场景穿插平民视角
