# 现代美国版三国 · 董王分支

> 董卓 = 唐纳德·特朗普

以《三国演义》精神为骨架，现代美国为舞台，参考《火凤燎原》叙事手法创作的政治军事史诗世界观。本分支将董卓直接映射为唐纳德·特朗普——一个被华盛顿建制派视为「野蛮人」的局外人，在帝国崩溃时以「让美国再次伟大」为口号闯入权力中心。

## 分支名含义

「董王」三重含义：
1. **董**——董卓的董
2. **王**——King（支持者称他为 God Emperor / The King）
3. **董王**——中文互联网「懂王」的谐音

## 故事梗概

> 「你们叫我小丑？你们叫我野蛮人？你们在华盛顿管理了这个国家五十年，结果呢？工厂搬走了，工作没了，边境开放了，华尔街把你们的钱都偷走了。我至少建了墙。」

2026年，加密货币「自由币」崩盘引发全美金融暴动，三千万人毕生积蓄化为乌有。这场危机不是意外——华尔街28岁的对冲基金经理司马懿通过 Mnemosyne Capital 做空获利三十亿美元，他的量化团队用算法制造了完美的流动性真空，为一个人的上台铺好了路。

那个人就是唐纳德·特朗普。

共和党建制派以为可以「招安」特朗普，利用他的基本盘赢回权力——就像何进召董卓入京诛杀宦官一样。他们以为自己可以控制他。2028年，特朗普以「抽干沼泽」「把工作带回来」为口号压倒性胜选。就职典礼上，建制派精英们站在他身后强颜欢笑，还没意识到他们召来的不是看门狗，是一头狮子。

就职后，特朗普清洗「不忠」官员，用行政命令绕开国会。2029年1月6日，数百万MAGA支持者聚集国家广场，部分支持者冲击国会山，国会大厦被纵火——旧政治共识在火焰中化为灰烬。特朗普的影子军师 Brother Wang——一个MIT金融博士、华尔街量化天才，从未出现在任何官方顾问名单上——站在白宫屋顶看着国会山方向升起的烟：「你烧的不是国会，是旧美国。」

两党建制派、主流媒体、科技巨头、司法部组成「反董联盟」。共和党党鞭王允表面合作，暗中策划连环计——他利用残兵间谍小孟（以「Celeste」身份渗透海湖庄园社交圈）同时接近特朗普和副总统，制造两人之间的裂痕。而在所有人视线之外，三个出身卑微的人正在崛起：宾州国民警卫队中校曹操、海岸警卫队少校孙坚、退伍军人事务部社工刘备。他们不是诸侯，不是将军，不是名人。但天下，终将是他们的。

## 核心人物

- **特朗普（董卓）**：纽约地产商/真人秀明星/总统，被精英视为野蛮人，用精英无法理解的逻辑颠覆所有规则
- **Brother Wang（李儒/许临）**：MIT金融博士→华尔街量化天才→特朗普影子军师，金融兵法，围棋高手，喜欢「先占个位置」
- **曹操**：宾州国民警卫队中校，速度兵法，从底层崛起
- **孙坚**：海岸警卫队少校，猛将，将缴获核武密码箱
- **刘备**：基层社工，人心兵法，退伍军人网络
- **司马懿**：28岁对冲基金经理，信息兵法，残兵组织幕后老板，设计棋盘的人
- **燎原火/赵云**：残兵首领，无痛觉刺客，以修车铺老板身份潜伏
- **小孟（貂蝉）**：跨性别女性，残兵渗透专家，与王允之女王蕊相恋
- **丁原**：共和党全国委员会主席，老牌建制派，「收」特朗普为义子
- **吕布**：副总统，丁原门生，被特朗普挖角，后因连环计背叛

## 叙事特色

- **计中计**：自由币崩盘是司马懿的局，特朗普上台是Brother Wang的局，建制派「招安」是所有人都以为自己在下棋的局
- **限知视角**：叙述者不解释谁在操控一切，读者自己拼凑
- **火凤式节奏**：董卓篇即独立完整三幕剧，不急着「三家分汉」
- **现实映射**：不换皮、不隐喻，特朗普就是特朗普——他本身就是三国式人物
