### 1.1 董卓：唐纳德·特朗普（Donald Trump）

| 维度 | 设定 |
|---|---|
| 全名 | 唐纳德·约翰·特朗普（Donald John Trump） |
| 出生 | 1946年6月14日，纽约皇后区 |
| 身份 | 特朗普集团董事长、《学徒》真人秀前主持人、美国第45任总统、第47任总统（2029年就职） |
| 外貌 | 七十八岁（2029年时），体态臃肿但精力旺盛。标志性的金发经过精心梳理但时常被风吹乱。橙色的肤色在电视镜头下更加明显。永远穿着过长的红色领带和过大的深色西装。握手时力道极大，喜欢拍对方的手臂和后背 |
| 说话方式 | 短句、重复、最高级形容词。不说完整句子，说"believe me"、"folks"、"let me tell you"、"nobody knows better than me"。激动时声音拔高，句子断裂，用手势代替完整表达。在正式演讲中使用提词器但经常偏离脚本，在集会上完全即兴——而后者才是真正的他 |
| 核心性格 | 永不认错。把每一次失败说成胜利，把每一个敌人说成失败者。极度敏感于任何对他的轻视——尤其是来自"精英阶层"的。直觉极强但不读书不看简报，靠看电视和打电话获取信息。对忠诚的要求是绝对的，但他自己对任何人的忠诚都是有条件的 |
| 被华府精英的看法 | 野蛮人、小丑、粗俗的暴发户、不可控的变量。华盛顿的鸡尾酒会上，人们用模仿他的说话方式来取乐。智库学者写文章分析他的"自恋型人格障碍"。共和党建制派和他保持距离但不敢公开决裂——因为他的支持者不是党员，是信徒 |
| 政治本质 | 他不是一个传统意义上的政治家。他是一个表演者，一个品牌，一个运动的符号。他的核心政治天赋不是制定政策，而是感知人群的情绪——他能闻到房间里的愤怒、恐惧、渴望，然后说那些话。他可能不理解政策细节，但他理解权力的气味 |
| 对应三国 | 董卓——被朝廷召入京城的边地军阀，所有人都以为能控制他，结果他控制了所有人 |

**角色深层解构：**

华盛顿的政治评论家们犯了一个根本性的错误——他们以为特朗普是一个可以被羞辱、被击败、被制度约束的人。他们不理解：一个在八十年代被整个纽约上流社会嘲笑为"暴发户"的人，一个经历过六次破产还能东山再起的人，一个被主流媒体骂了十年还能赢得大选的人，他的心理结构不是正常人的心理结构。

正常人被全网嘲讽会崩溃。特朗普被全网嘲讽会发推文。

正常人输了选举会承认失败。特朗普输了选举会说选举被窃取了——而且他真的相信这一点，或者说，他已经分不清"我相信"和"对我有利"之间的区别了。

这是他最大的弱点，也是他最大的力量。他的支持者爱他，不是因为他正确，而是因为他从不低头。在一个所有政客都在道歉、解释、妥协的时代，一个从不道歉的人有一种原始的吸引力。

他在深夜打电话。不是给顾问——他的正式顾问名单上的人，他在白天见。深夜的电话打给那些不在名单上的人。福克斯新闻的主持人。脱口秀电台的主持人。各州的共和党主席。捐钱的亿万富翁。还有一个人——一个他从不向任何人提起的人。那个人的声音很轻，说话很慢，从来不叫他"总统先生"，也从来不叫他"唐纳德"。那个人叫他"董先生"。

特朗普不知道那个人的真名。他只知道那个声音，以及那个声音给他的建议——那些建议总是对的。

**经典场景·椭圆形办公室的深夜：**

2029年1月22日，凌晨两点十七分。

椭圆形办公室的灯还亮着。特朗普坐在坚毅桌后面，面前摊着一堆打印出来的推特评论——他不用幕僚准备的舆情报告，他要看"真正的人"在说什么。电视调到福克斯新闻，静音了，但画面一直在他眼角余光里跳动。

桌上的红色电话响了。不是国安热线，是他的私人手机——一部普通的安卓手机，被特勤局反复检查过，但他坚持要用自己的手机。

"喂。"

"董先生。"那个声音。总是这么轻，这么慢，像在棋盘上下子。

"王！你看到今天的收视率了吗？就职演说，四千九百万。四千九百万！史上最高。奥巴马那个家伙才三千八百万，他们还说他是明星。"

"看到了。"

"你觉得我演讲怎么样？Bully，对吧？真的bully。那个'抽干沼泽'的段落，现场都疯了。"

"很好。但有一件事更重要。"

"什么事？"

"彭斯。"

特朗普皱眉。他不喜欢在心情好的时候谈不愉快的事。"彭斯怎么了？他是副总统，他做好副总统该做的事就行了。祈祷，什么的。"

"他今天上午去了圣约翰圣公会教堂。不是一个人去的。和麦康奈尔一起。"

特朗普的脸沉了下来。麦康奈尔。参议院多数党领袖。那个老乌龟。他还没有来白宫拜谒过。

"他们谈了什么？"

"不知道。但他们在教堂里待了四十七分钟。祈祷不需要四十七分钟。"

沉默。特朗普把手里的打印纸揉成一团。"我选他是因为那些基督徒喜欢他。他知道自己该站在哪一边。"

"他知道自己站在哪一边。问题是，他觉得那一边是不是你这边。"

"你什么意思？"

"丁原还在。"

丁原。共和党全国委员会主席。那个把政党资源给他的人。那个在初选中帮他清除其他候选人的人。那个在他当选后还在试图"引导"他的人。

"丁原是个好人，"特朗普说，但语气像在背台词，"他一直支持我。"

"他支持你，是因为他以为能控制你。现在他发现控制不了了。他在找退路。彭斯是他的退路。"

"那你说我该怎么办？"

"什么都不做。现在还不是时候。先让他动。人在以为自己有退路的时候，才会犯错。"

电话挂了。特朗普看着手机屏幕上"通话结束"四个字，然后把手机扔到桌上。他拿起遥控器，把福克斯新闻的声音打开。主持人正在重播他的就职演说片段。

"史上最高收视率，"他对着空无一人的办公室说，"史上最高。"

### 1.2 Brother Wang：王博士（Dr. Wang）

| 维度 | 设定 |
|---|---|
| 代号/称呼 | Brother Wang（特朗普身边极少数人这样叫他）、王博士（华尔街旧识）、"那个中国人"（情报机构档案中）、军师（他自己从不这样自称） |
| 真名 | 无人知晓。在MIT读博时用的名字是"Wang Wei"，但这几乎可以肯定不是真名。联邦调查局的档案中列有十七个疑似化名 |
| 出生 | 推测为1980年前后（MIT博士毕业于2005年，年龄约25-28岁）。出生地不详，可能是中国内地某个城市，可能是香港，可能是新加坡——口音是标准美式英语，但偶尔会蹦出几句带有北京儿化音的中文 |
| 外貌 | 四十五岁左右（2029年），亚裔男性，身高约一米七五，瘦削，永远穿深色定制西装但从不打领带。戴一副无框眼镜，镜片后面的眼睛永远半眯着，像在打盹又像在算牌。头发花白但修剪整齐。左手无名指上戴着一枚极其普通的银色戒指——不是婚戒，没有人知道那枚戒指的来历 |
| 教育背景 | MIT斯隆管理学院金融学博士，2005年毕业，博士论文题目是《论信息不对称下的市场崩盘机制：一个博弈论模型》。他的导师后来获得了诺贝尔经济学奖，但在推荐信中只写了一句话："此生不需要推荐信。" |
| 华尔街经历 | 2005-2015年，在文艺复兴科技公司（Renaissance Technologies）担任量化策略师，是该公司历史上最年轻的高级策略师之一。2015年突然离职，没有给出任何理由。离职时他管理的基金年化收益率是78%——比西蒙斯本人还高。离职后他消失了两年 |
| 与特朗普的关系 | 2017年特朗普上任第一年，通过特朗普集团的一个商业地产项目间接接触到特朗普家族。2019年开始直接与特朗普通话——最初是关于关税政策对汇率的影响，后来话题越来越广。他从未出现在任何官方顾问名单上，从未在白宫西翼有过办公室，从未接受过任何媒体采访。华盛顿没有人知道他的存在——直到2029年之后很久 |
| 核心能力 | 金融兵法。他把金融市场看作一个巨大的信息战场——价格是战报，成交量是兵力部署，波动率是士气，流动性是补给线。他的交易策略不是技术分析，不是基本面分析，而是博弈论——他在算的不是市场会怎么走，而是其他参与者以为市场会怎么走，然后提前两步布局。围棋高手——业余七段，在纽约的围棋俱乐部里下棋时从不用真名，棋风被评价为"像水一样，你以为挡住了他，他已经从别的地方流过去了" |
| 兵法哲学 | **「混沌兵法」**——"秩序是给弱者的安全毯。强者在混沌中取利。最完美的计划不是让一切按计划发生，而是让一切看似意外地发生。" |
| 对应三国 | 李儒 + 贾诩的混合体——董卓的首席军师，但比李儒更阴、更隐、更长远。他不只是董卓的军师，他是整个棋局中最早看清全局的人之一 |

**角色深层解构：**

Brother Wang是那种"不存在的人"。他没有社交媒体账号。他的照片在互联网上找不到——唯一一张模糊的照片是2013年在一次量化金融会议上被参会者无意中拍到的，照片里他低着头看手机，只能看到半张脸。他在纽约中央公园附近有一套公寓，但邻居说他一年最多在那里住一个月。

特朗普身边的人知道他的存在——但知道得很少。伊万卡叫他"爸爸的那个中国朋友"。库什纳不信任他，因为查不到他的底细。班农在2017年见过他一次，后来在《战室》 podcast上说过一句话："那个人的眼睛——你知道吗——就像两口枯井。你往里看，看不到底。"

Brother Wang不追求权力，不追求财富，也不追求名声。他追求的东西更难理解——他追求的是"验证"。他的博士论文提出了一个关于人类社会在极端压力下如何崩溃的数学模型。华尔街给了他验证这个模型的实验室，但市场只是社会的一个子系统。特朗普的崛起给了他一个更大的实验室——一个完整的国家、完整的政治体系、完整的文明。

他在看着。他在计算。他在下子。

他不是任何人的傀儡。他为特朗普工作，是因为特朗普是这个时代最有趣的变量。当特朗普不再有用的时候，他会像丢掉一颗废子一样丢掉他。

但那是很久以后的事了。现在，棋盘上的中盘才刚刚开始。

**经典场景·海湖庄园的棋桌：**

2028年8月，共和党全国代表大会前两周。

海湖庄园的露台。南佛罗里达的湿热空气像一块湿毛巾裹在人身上。Brother Wang坐在一张大理石桌前，面前摆着一副围棋棋墩。他一个人在打谱——吴清源对本因坊秀哉的"星三三·天元"之局，1933年。那盘棋改变了围棋。

脚步声。他没有抬头。

"王先生。"

是库什纳。贾里德·库什纳。特朗普的女婿。穿着定制的亚麻衬衫，扣子扣到最上面一颗，即使在这种天气里也不出汗——或者说，他出汗但不让人看到。

"库什纳先生。"Brother Wang落了一子。白子。棋盘上的局势看起来很散，但如果懂棋的人看，会发现白子的势力像一张网，在看不见的地方连在了一起。

"我查过你。"库什纳在他对面坐下。

"我知道。"

"你的MIT记录是真的。但你在文艺复兴之前的经历——不存在。没有出生证明，没有高中记录，没有社保号。2002年之前你这个人不存在。"

"2002年之前我在别的地方。"

"哪里？"

"库什纳先生，"Brother Wang终于抬起头，镜片后的眼睛半眯着，"你来找我，不是为了问我的出身。你是来问我为什么你岳父的民调突然领先十二个点。"

库什纳的表情没变。这是他在谈判桌上练了二十年的本事——永远不让对手看到你被说中了。但Brother Wang不需要看他的表情。他看的是库什纳放在桌上的手——左手食指无意识地敲击了两下桌面。那是焦虑。

"是你做的？"库什纳问。

"你觉得呢？"

"自由币崩盘的时候，有一笔三十亿美元的做空收益。我们追踪了那笔钱——经过十七层壳公司，最后流入了一个在开曼注册的基金会。那个基金会的钱，有一部分流进了竞选的超级PAC。"

"一部分。"Brother Wang纠正他，"是很小的一部分。大部分钱还在市场里。"

"你在操纵选举？"

"我在创造条件。操纵是告诉人们该投给谁。创造条件是让人们自己想投给他。区别很大。"

"怎么做的？"

Brother Wang微微一笑。这是一个罕见的表情——不是因为开心，而是因为他接下来要解释的东西，对他来说像呼吸一样简单，但对面前这个人来说可能永远理解不了。

"库什纳先生，你做房地产。你知道什么最重要？location, location, location。在我的行业里，最重要的是narrative, narrative, narrative。叙事。自由币崩盘之后，三千万人失去了积蓄。他们愤怒。他们需要一个解释。建制派告诉他们：'这是市场调整。'华尔街告诉他们：'你们应该做更多研究。'美联储告诉他们：'我们会降息。'你岳父告诉他们：'是他们偷了你们的钱。我会帮你们抢回来。'"

"然后呢？"

"然后我要做的，就是确保这句话在他们的脸书、推特、TikTok、YouTube上出现。不是一次。不是十次。而是每一次他们打开手机，都能看到这句话。算法会帮我——因为算法的逻辑很简单：人们点击什么，就推送什么。愤怒是最好的点击率。我只需要在最初的几个关键节点投入资源，让愤怒的雪球滚起来。之后，雪球自己会滚。"

"社交媒体公司不会阻止你吗？"

"他们阻止不了。因为他们的算法和我的算法在做同样的事——最大化engagement。我只是比他们更懂人性。"

库什纳沉默了很久。海风从大西洋吹过来，把棕榈树吹得沙沙响。

"你想要什么？"库什纳最终问。这才是他真正的问题——不是"你做了什么"，而是"你为什么要做"。

"一盘棋。"Brother Wang说。

"什么？"

"一盘有趣的棋。这盘棋在华尔街下了十年，我赢了，但是棋盘太小。现在棋盘是整个国家。可能是整个世界。"

"你把美国的政治当棋下？"

"一切都是棋。区别只是棋盘的大小和赌注的高低。"

库什纳站起来。他不喜欢这个人。他不信任这个人。但他不得不用这个人——因为他岳父需要这个人。

"总统不能知道你做的所有事。"

"他不知道。他不需要知道。他只需要知道结果。"

"如果他问呢？"

"告诉他这是他的直觉。他会相信的。"

库什纳走了。Brother Wang继续打谱。棋盘上的白子已经布成了一个经典的布局——"中国流"。吴清源的棋。改变了围棋的棋。

他落了最后一子。白子。中腹。看似无关紧要的一子。但在三十手之后，这一子将决定整盘棋的走向。

他看着棋盘，轻声说："落子无悔。"

### 1.3 李儒：贾里德·库什纳（Jared Kushner）

| 维度 | 设定 |

**角色深层解构：**

库什纳是那种在每个政权里都会存在的人——不是最聪明的，不是最勇敢的，不是最忠诚的，但他是那个"做事"的人。当其他顾问在争论意识形态的时候，库什纳在打电话。当其他顾问在推特上骂战的时候，库什纳在签合同。

他有一种罕见的能力：他能让自己消失在背景里。在特朗普政府的内阁会议上，他坐在角落里，不说话，不抢镜头，但会后他是那个去找每个关键人物"单独聊聊"的人。外国元首和他打交道时会觉得他很理性、很温和、很好说话——直到他们发现，所有他们以为"库什纳已经同意了"的事情，最后都没有兑现，因为库什纳的"我会考虑"就是"不"的意思。

他对特朗普的感情是复杂的。一方面，他欠特朗普——特朗普给了他一个地产世家子弟永远不可能靠自己得到的舞台。另一方面，他看不起特朗普的粗俗、冲动、不可预测。他在岳父身上看到了一种原始的权力，但也看到了一种原始的危险。他像一个在骑老虎的人——他知道自己不能下来，但他也不确定自己能骑多久。

2028年大选之后，他开始准备后路。不是背叛——他还没有到那一步。但他开始在中东做更多的"个人投资"，在沙特和阿联酋建立自己的关系网。他知道政治权力是暂时的，但关系网是永久的。

Brother Wang知道这一切。他没有告诉特朗普。因为库什纳准备后路这件事，本身就是Brother Wang计划的一部分。一个没有退路的人是危险的——但一个自以为有退路的人，更容易被操控。

**经典场景·白宫西厢办公室：**

2029年2月，就职后第三周。

库什纳的办公室在白宫西翼二楼，比椭圆形办公室小得多，但视野更好——能看到整个南草坪。他在这个办公室里待了四年（第一任期），现在又回来了。墙上挂着一幅亚伯拉罕·林肯的肖像——不是因为他崇拜林肯，而是因为林肯是唯一一个"在分裂的国家中维持了联邦"的总统，而库什纳需要每天提醒自己这个任务的难度。

门被敲响了。

"进来。"

是伊万卡。她穿着一件白色的连衣裙，金色的头发盘在脑后，手里拿着两杯咖啡。

"你还没睡？"她把一杯咖啡放在他桌上。

"睡不着。"

"又是因为王？"

库什纳抬头看她。伊万卡比她父亲聪明——比她父亲身边几乎所有人都聪明。她有一种在社交场合中瞬间判断局势的天赋，这是她父亲靠直觉做到的事，但她靠的是冷静的观察和计算。

"他在做一些我不知道的事。"库什纳说。

"他一直在做你不知道的事。爸爸听他的。这不是新闻。"

"这不一样。竞选的时候，他做的是社交媒体、算法、资金流向——那些我都知道，虽然我不知道细节。但现在……"他停顿了一下，"现在他在和一些人接触。军方的人。情报界的人。一些不应该和他接触的人。"

"你怎么知道？"

"我让人留意了。"

伊万卡皱眉。"你在监视他？"

"我在留意。"库什纳纠正，"区别很大。"

"你觉得他在策划什么？"

"我不知道。这正是我担心的。如果他在为爸爸做事，我应该知道。如果他在为自己做事……"他没有说完这句话。

伊万卡走过来，坐在他办公桌的边缘。这是她在私人时刻才会做的事——不是第一女儿，不是白宫顾问，只是他的妻子。

"贾里德，"她说，声音很轻，"你有没有想过，也许爸爸知道他在做什么？"

"你爸爸连情报简报都不看。他靠看电视获取信息。"

"但他的直觉——"

"他的直觉让他赢了选举。但治理一个国家不是靠直觉。"

沉默。窗外，南草坪的灯在夜色中发着冷白的光。

"1月6号的事，"伊万卡突然说，"你觉得那是意外吗？"

库什纳的手停住了。他端起咖啡杯，但没有喝。1月6号。就职典礼前两周——不对，是就职日当天的国会认证。不，等一下——在这个时间线里，1月6日是2029年，是特朗普就职之后的事。火烧洛阳。

"你看到了什么？"库什纳问。

"我看到爸爸在那个早上发了一条推文。'来华盛顿。会很疯狂。'然后百万人就来了。你觉得那是即兴的吗？"

"你爸爸的推文都是即兴的。"

"那一条不是。我看到他发推文之前在打电话。打了七分钟。"

"打给谁？"

"我不知道。他用的是那部私人手机。通话记录里显示是'未知号码'。"

库什纳放下咖啡杯。咖啡已经凉了。他看着窗外的黑暗，脑子里在拼图——Brother Wang的暗中接触、1月6号的推文、彭斯和麦康奈尔在教堂的会面、丁原最近和众议院议长的几次"私人午餐"。

"我们需要做两手准备，"他最终说。

"什么意思？"

"如果一切顺利——爸爸控制了局面，王的计划成功了，我们继续做我们该做的事。如果出了问题……"他看着伊万卡，"我们需要确保我们自己是安全的。"

"你是说……背叛他？"

"我是说，确保我们有选择的余地。"

伊万卡看着他。她认识这个男人二十年了。她知道他不是一个理想主义者，不是一个斗士，不是一个信徒。他是一个幸存者。在一个即将被火焰吞没的城市里，幸存者会先找到出口在哪里——不是因为他想逃跑，而是因为他知道，如果没有出口，所有人都会被烧死。

"小心点，"她说，"王如果发现你在查他——"

"他知道。"库什纳说。

伊万卡愣住了。

"他知道我在留意他。他可能比我更早知道。但他什么都没做。"

"为什么？"

库什纳露出一个疲惫的微笑。"因为他不怕。一个不怕被监视的人，要么是清白的，要么是……"

"要么是什么？"

"要么他已经准备好了一切，即使你看到了他在做什么，你也阻止不了他。"

### 1.4 丁原：共和党全国委员会主席

| 维度 | 设定 |
|---|---|
| 姓名 | 罗伯特·"鲍勃"·丁沃尔（Robert "Bob" Dingwall）——华府政治圈内人称"丁原"（这个外号最早来自他的亚裔幕僚，后来因为太贴切而流传开来） |
| 身份 | 共和党全国委员会（RNC）主席，共和党建制派的核心人物，在华府政治圈摸爬滚打四十年的老狐狸 |

| 背景 | 俄亥俄州小镇出身，父亲是钢铁工人，靠奖学金读完大学，从县党部志愿者做起，一步一步爬到RNC主席的位置。经历过尼克松、里根、老布什、小布什、特朗普——五任共和党总统，他见过所有类型的政治家，或者说，他以为自己见过 |

| 与特朗普的关系 | 2016年他最初反对特朗普——和所有建制派一样，他认为特朗普是个笑话，会在大选中惨败。但特朗普赢得了初选，也赢得了大选，丁原作为RNC主席不得不和他合作。2020年特朗普败选后，丁原试图"回归正常"——支持了其他候选人，但特朗普的支持者惩罚了那些"背叛者"。2024年自由币崩盘后，当特朗普宣布参加2028年大选时，丁原做了一个他认为"理性"的决定：既然阻止不了他，就"收编"他。给他政党资源、竞选资金、党机器的支持——作为交换，他要特朗普承诺"执政时遵循保守派原则"，并选择一个"合适的"副总统。他以为这是"招安"。他以为他控制了候选人的选择——彭斯式的副总统是他推荐的。他以为，就像历史上所有党主席一样，他是那个"造王者"。他错了 |

**角色深层解构：**

丁原的悲剧在于他是一个"旧时代的人"。他理解的政治是委员会房间里的政治、筹款晚宴上的政治、党团会议里的政治。在那个世界里，关系、资历、交易和承诺是货币。他以为特朗普也在这个游戏里——一个更加粗俗、更加不可预测的玩家，但仍然是一个玩家。

他不理解的是，特朗普根本不在玩他的游戏。特朗普在玩一个完全不同的游戏——一个关于群众情绪、媒体叙事、个人崇拜的游戏。在这个游戏里，政党机器不是盟友，是障碍；政治承诺不是契约，是工具；传统保守派原则不是信仰，是可以随时抛弃的累赘。

丁原发现这一点的时候，已经太晚了。2028年大选胜利后，特朗普没有"感谢党"。他在胜选演讲中说："我不是共和党的总统。我是你们的总统。"——这句话直接否定了丁原四十年政治生涯的核心信念。

就职之后，丁原发现RNC的资金渠道被绕过了——特朗普的竞选团队直接向支持者募捐，不需要党部的中介。RNC的人事任命被架空了——特朗普把自己的人安插进了全国各州的党部。RNC的政策制定权被无视了——特朗普用行政命令而不是国会立法来施政，根本不需要党的立法议程。

丁原开始反击。他秘密接触彭斯。他秘密接触麦康奈尔。他秘密接触民主党领袖。他以为自己是在"拯救共和党"。他以为自己还是那个能在烟雾缭绕的房间里决定国家走向的人。

但他忘记了一件事：在政治中，如果你已经失去了群众，你就什么都不是。他在华府的人脉网络是一张漂亮的蜘蛛网——但特朗普带来的是一场飓风。

**经典场景·国会山的秘密会议：**

2029年3月，海湖庄园"郿坞"时期之前。

国会山地下，RNC总部的一间没有窗户的会议室。丁原把拐杖靠在桌边，艰难地坐下来。房间里还有三个人：麦康奈尔（参议院多数党领袖）、约翰逊（众议院议长）、彭斯（副总统）。

彭斯坐在最远的角落，面前的咖啡一口没动。他看起来比就职前瘦了——脸颊凹陷，眼睛下面有深重的黑眼圈。副总统这个职位本来就没有什么实权，但在特朗普政府里，它比没有实权更糟——它是一个镀金的笼子。

"他今天又签了三个行政命令，"丁原说，声音沙哑但稳定，"三个。没有经过委员会审议，没有经过法律顾问审核，直接签了。一个是关于关税的，一个是关于边境的，第三个——"他停顿了一下，"第三个是关于'国内叛乱分子'的定义。"

"什么定义？"麦康奈尔问。他的声音永远是那种慢吞吞的肯塔基口音，让人分不清他是在思考还是在打瞌睡。

"他把'任何试图以非法手段阻挠联邦政策执行的组织和个人'定义为国内叛乱分子。然后授权国土安全部和司法部采取'一切必要手段'。"

沉默。房间里的空调嗡嗡作响。

"那包括我们吗？"约翰逊问。他是新任议长，比丁原年轻二十岁，但看起来已经被这份工作压垮了。

"如果我们反对他，就包括我们。"丁原说。

"我们不能让他这么做，"约翰逊说，"国会有权——"

"国会有什么权？"丁原打断他，声音里第一次有了疲惫和苦涩，"他在众议院有多少席位？二百二十一个。其中至少五十个是他的人——MAGA派，他们效忠的不是党，是他个人。参议院呢？一百个席位里，我们有五十一席，但至少二十席不敢投反对票——因为他们害怕初选。我们能弹劾他吗？能。但参议院能定罪吗？"

他看向麦康奈尔。麦康奈尔没有说话。他的沉默就是答案。

"还有媒体，"丁原继续说，"福克斯已经完全是他的了。Newsmax、OANN更不用说。社交媒体上，他的粉丝有九千万。九千万。我们发一份声明，CNN会报道——但他发一条推文，九千万人直接看到。"

"司法呢？"彭斯突然开口。这是他今晚说的第一句话。所有人都看向他。"最高法院。我们有六比三的保守派多数。他们可以裁定行政命令违宪。"

丁原看着他。彭斯是他推荐的副总统人选。一个虔诚的基督徒，一个坚定的保守派，一个按照传统路线一步步爬上来的政治家——丁原以为彭斯会是"缰绳"，会是那个在特朗普失控时拉住他的人。但现在彭斯坐在角落里，像一只受惊的鸽子。

"迈克，"丁原温和地说，"最高法院裁定违宪需要有人起诉。起诉需要时间。而在这段时间里，他的行政命令已经在执行了。而且——"他犹豫了一下，"而且最高法院的首席大法官上周和他在白宫吃了晚饭。"

彭斯的脸白了。

"你怎么知道？"

"我在白宫有人。"丁原说。他没有说那个人是谁。在政治中，消息来源比消息本身更珍贵。

"那我们怎么办？"约翰逊问，声音里有了一丝恐慌。

丁原环顾了一圈房间。三个男人。三个在华府权力体系中爬到顶峰的男人。三个加起来有超过一百年政治经验的男人。他们在这个没有窗户的房间里密谋，像阴谋家——但他们曾经是这个体系的主人。

"我们等，"丁原说，"他会犯错的。他这种人一定会犯错。他太自信了，太冲动了，太相信自己的直觉了。当他犯错的时候——当他真正惹怒这个国家的时候——我们就动手。"

"怎么动手？"麦康奈尔问。

"弹劾。第二十五修正案。必要的时候……"他没有说完。但房间里每个人都明白他的意思。

彭斯站起来。"我不能参与这个，"他说，声音在发抖，"我是副总统。我在圣经面前宣誓过——"

"迈克。"丁原叫住他。他站起来，拄着拐杖走到彭斯面前。近距离看，这位老人的眼睛里有一种彭斯从未见过的东西——不是愤怒，不是恐惧，而是一种深沉的、历经沧桑的悲伤。"迈克，你在圣经面前宣誓的是捍卫宪法。不是捍卫一个人。"

彭斯看着他。然后他低下头，走出了房间。门在他身后关上，发出沉闷的声响。

麦康奈尔等门关上后才开口："他会叛变吗？"

"不会，"丁原说，"他太虔诚了，不会主动叛变。但当关键时刻到来的时候——当他必须选择站在哪一边的时候——他会选择宪法。"

"你确定？"

丁原回到桌边，慢慢坐下。他拿起面前已经凉了的咖啡，喝了一口。

"我不确定，"他说，"但这是我们手上唯一的牌了。"

### 1.5 吕布：副总统迈克·彭斯（Mike Pence）

| 维度 | 设定 |
|---|---|
| 全名 | 迈克尔·理查德·彭斯（Michael Richard Pence） |
| 身份 | 美国第48任副总统、前印第安纳州州长、前众议院议员 |
| 年龄 | 六十九岁（2029年），白发，总是梳得整整齐齐。面容严肃，下颌方正，像一尊从50年代走出来的保守派雕像。永远穿深色西装、白色衬衫、红色或蓝色领带，领带结打得一丝不苟 |
| 背景 | 印第安纳州出身，天主教徒后来改信福音派，在电台脱口秀主持人的位置上开始政治生涯。以"基督教保守派战士"的身份当选众议员，后任印第安纳州州长。2016年被特朗普选为竞选搭档——不是因为特朗普喜欢他，而是因为他能带来福音派的选票。2028年再次成为竞选搭档，原因相同 |
| 核心性格 | 虔诚、自律、拘谨、道德感极强。他每天早上读圣经，每周日去教堂，不在没有妻子在场的情况下和其他女性单独用餐（著名的"彭斯规则"）。他相信自己是一个"有原则的人"——但在政治中，原则有时候是盔甲，有时候是枷锁。他说话很慢，每个字都像是在心里掂量过才出口。他从不说脏话，从不在公开场合失态，从不即兴发言——他的每一次演讲都是照着稿子念的 |
| 与丁原的关系 | 政治上的父子。丁原在彭斯还是众议员的时候就注意到了他——一个可靠的、忠诚的、能守住保守派底线的年轻人。丁原在2016年向特朗普推荐了彭斯，2028年又一次推荐。彭斯对丁原怀有深深的敬意——丁原代表了"老共和党"的那种稳重和纪律，而彭斯认为那是共和党应该有的样子 |
| 与特朗普的关系 | 复杂。他对特朗普有一种近乎本能的反感——特朗普的粗俗、好色、不虔诚、对传统的蔑视，都让他不舒服。但他也对特朗普有一种政治上的敬畏——特朗普赢了，而特朗普赢的方式是彭斯永远做不到的。他像一个被安排嫁给野蛮部落首领的公主——他履行妻子的义务，但他不爱他的丈夫，也不信任他 |
| 内心冲突 | 他是一个真正的信徒——相信上帝，相信宪法，相信美国的制度。但他也是一个副总统——他的全部政治权力都来自于一个他不信任的人。当1月6日到来的时候，当百万MAGA支持者要求他"做正确的事"的时候——这个"正确的事"到底是什么？是支持总统？还是捍卫宪法？他的信仰告诉他要服从权柄，但他的誓词告诉他要捍卫宪法。这两者在他的灵魂里撕裂 |
| 对应三国 | 吕布——丁原的义子，后投奔董卓，但最终背叛董卓。但本分支中的吕布不是武艺天下第一的猛将，而是一个内心被信仰和忠诚撕裂的副总统。他的"背叛"不是因为野心或美色，而是因为良心 |

**角色深层解构：**

彭斯是这个故事里最悲剧的人物之一——不是因为他强大，而是因为他软弱；不是因为他有野心，而是因为他没有野心。他不想当总统。他从来都不想当总统。他想当的是一个好副总统，一个好基督徒，一个好的保守派战士。但历史把他放在了一个他没有准备好的位置上。

他和小孟（Celeste）的相遇是王允连环计的关键。小孟以"海湖庄园的新社交名流"身份出现时，彭斯正处于他人生中最孤独的时刻——被特朗普冷落、被建制派拉拢、被自己的良心折磨。小孟的倾听、理解、温柔（以及精心设计的"共同信仰"伪装——小孟在福音派家庭长大，知道怎么说基督徒的语言），对彭斯来说是一种他在华盛顿从未体验过的慰藉。

他不知道这一切都是假的。他不知道那个在烛光下和他一起祷告的女人，是一个被训练来毁灭他的刺客。

他的"背叛"将不是一个戏剧性的瞬间——不是他在参议院里站起来大喊"我反对总统"。那不是彭斯的风格。他的背叛将是一个沉默的、痛苦的、拖延了很久的决定——就像2021年1月6日他最终决定认证选举结果一样。他会祈祷很多天，会和牧师谈很多次，会在椭圆形办公室里和特朗普有一次漫长而尴尬的会面，然后他会做他认为"上帝要他做的事"。

而那件事，将是董卓阵营崩溃的开始。

## 第二章 竞选之路——懂王分支叙事

> 本章接续共通文档2.3节"宪法危机"之后的叙事。共通文档描述了2027年初的宪法僵局：总统被25修正案程序困住，副总统拒绝签字，全国陷入宪法无人区。懂王分支的故事从这里开始——不是以军事坦克进入华盛顿的方式，而是以一场选举的方式。

### 2.1 黑暗中的棋手：Brother Wang的布局

2027年1月，华尔街。

Mnemosyne Capital的办公室在曼哈顿中城一栋玻璃幕墙大楼的五十八层。从落地窗望出去，整个中央公园像一块巨大的墨绿色地毯铺在城市中央。Brother Wang站在窗前，手里端着一杯乌龙茶——不是英国红茶，不是咖啡，是一个旧搪瓷缸子泡的乌龙，和这间价值八千万美元的办公室格格不入。

他的首席交易员，一个叫帕克的韩裔美国人，站在他身后，看着三块屏幕上滚动的数据。

"自由币的流动性已经枯竭了百分之八十七，"帕克说，"但还有人在接盘。"

"谁？"

"主要是散户。还有一些中小基金在抄底。他们觉得这是底。"

"让他们觉得。"Brother Wang喝了一口茶，"再放三亿美元进去买。"

帕克愣了一下。"买？我们不是在做空吗？"

"先买。让价格反弹百分之十五。让媒体写'自由币触底反弹'、'华尔街开始抄底'。让那些抄底的人觉得他们对了。然后——"他停顿了一下，"然后在他们加仓的时候，再砸。"

"砸多少？"

"全部。"

帕克倒吸一口冷气。"那会引发——"

"恐慌。挤兑。血洗。"Brother Wang的声音没有任何波动，像在说天气预报，"自由币的底层架构是去中心化的——这意味着没有央行来救市。没有熔断机制。没有跌停板。当恐慌开始的时候，它会归零。不是几乎归零——是零。"

"三千万人——"

"会失去他们的积蓄。"Brother Wang转过身来，看着帕克。镜片后面的眼睛在屏幕蓝光的映照下像两块冰冷的石头。"帕克，你知道革命是怎么发生的吗？不是因为人们穷。穷了几千年了，人们习惯了穷。革命发生在人们以为自己会变富，但突然发现自己变穷了的时候。希望破灭的时候，才是愤怒爆发的时候。"

"我们是在制造一场革命？"

"我们是在创造一个机会。"Brother Wang走回办公桌，桌上摆着一副围棋——不是装饰，是真正在用的棋具，棋罐里的棋子已经被摸得发亮。他拈起一颗黑子，落在棋盘的星位。"自由币崩盘之后，这个国家会陷入混乱。混乱中，人们需要一个强人。一个告诉他们'我能解决问题'的人。建制派解决不了问题——因为他们就是问题本身。华尔街解决不了问题——因为他们是小偷。媒体解决不了问题——因为他们是骗子。"

"特朗普。"帕克说出了那个名字。

"特朗普。"Brother Wang落了第二颗黑子，在对角的星位。"他不是最好的候选人。他甚至不是一个好人选。但他是唯一一个建制派控制不了的人。唯一一个能让那些愤怒的人觉得'他在为我说话'的人。"

"你觉得他能赢？"

Brother Wang微微一笑。他没有回答这个问题。他落了第三颗黑子——不是在角上，不是在边上，而是在中腹。一个奇怪的位置。在围棋里，过早落子中腹通常是新手的错误。

但帕克不是棋手。他看不到这颗看似随意的黑子，和三十手之后的杀棋之间的关系。

"帕克，"Brother Wang说，"你知道我为什么喜欢围棋而不是国际象棋吗？"

"不知道，王博士。"

"国际象棋的目标是杀死国王。所有的棋子都是工具——兵、马、象、车，它们的价值是固定的，它们的角色是固定的。一个兵就是一个兵，永远变不成皇后。但围棋不一样。围棋里没有国王。每一颗棋子在落下的时候都是平等的。一颗棋子的价值取决于它在哪里，和谁在一起，在什么时机落下。一颗看起来要死的废子，可以在三十手之后成为杀棋的关键。"

他看着棋盘上的三颗黑子。

"特朗普是那颗中腹的废子。所有人都觉得他没有价值。所有人都觉得他早该死了。但在这个棋局里——"他抬起头看着帕克，"——废子往往是决定胜负的手。"

帕克走后，Brother Wang一个人在办公室里待了很久。窗外，曼哈顿的天际线在暮色中亮起万家灯火。他拿起手机，拨了一个号码。

电话响了三声，然后接通了。

"喂？"那个声音。沙哑的，不耐烦的，带着一种被打扰的恼怒。

"董先生。"

"王！你他妈知道现在几点吗？"

"知道。但你应该听听这个。"

"说。"

"三周之内，会有一场金融危机。比2008年大。"

电话那头沉默了两秒。然后是一声嗤笑。"你在逗我？市场现在好得很。道指刚破了三万八。"

"三周。你会看到的。"

"所以呢？你给我打电话就是为了告诉我天要塌了？"

"我打电话是要告诉你，当天塌下来的时候，你要在推特上发一句话。"

"什么话？"

"'我早告诉过你们。'"

沉默。然后特朗普笑了——不是他在集会上那种夸张的笑，而是一种低沉的、发自内心的、像鲨鱼闻到血腥味一样的笑。

"然后呢？"

"然后等着。会有人来找你的。"

"谁？"

"那些觉得他们能控制你的人。"

电话挂了。Brother Wang把手机放在桌上，继续打谱。窗外，纽约的灯火像一片星海。他落了第四颗黑子——在边星的位置，一个稳健的防守手。

在他身后的屏幕上，自由币的价格在绿色和红色之间跳动。三千万人的命运，就在这些红绿数字之间被决定了。没有人知道。甚至那些即将失去一切的人自己也不知道。

这就是混沌兵法的第一课：最好的计划，是让所有人都以为一切都是意外。

### 2.2 何进召董卓：建制派的"招安"

2027年3月，华盛顿。

自由币崩盘了。

事情的发展比Brother Wang预期的还要猛烈。自由币从38000美元的高点跌到了23美元——不是三周，是九天。三千万美国人的毕生积蓄，在不到两周的时间里灰飞烟灭。全美200多个城市爆发了暴动。总统签署了《紧急地方授权法案》，但国民警卫队在很多城市根本控制不了局面——因为国民警卫队的士兵们自己也有积蓄在自由币里。

华府陷入了恐慌。

在国会山一间镶着橡木护墙板的会议室里，共和党高层召开了一次紧急会议。没有会议记录，没有工作人员在场，只有七个人：RNC主席丁原、参议院多数党领袖麦康奈尔、众议院议长（即将被罢免的那位）、参议院少数党领袖（民主党人舒默——这次危机迫使两党暂时合作）、白宫幕僚长莫雷诺、以及两位前总统（小布什和奥巴马，他们是作为"党国元老"被秘密请来的）。

"我们控制不了局面了，"莫雷诺说。她看起来像是一周没睡觉——眼睛浮肿，咖啡杯在手里发抖。"总统已经完全——他已经做不了决定了。他在椭圆形办公室里看福克斯新闻，谁的电话都不接。"

"那25修正案呢？"小布什问。还是那个慢吞吞的德克萨斯口音，但眼神比他当总统时锐利得多——离任后的人总是看得更清楚。

"副总统不签字。"

"为什么？"

"他说——"莫雷诺停顿了一下，像是在回忆一句让她不舒服的话，"他说'现在不是时候。国家需要团结，不是更多的混乱。'"

"放屁，"麦康奈尔说。这是他今晚说的第一句话。"他不是怕混乱。他是怕自己成为替罪羊。如果他签了25修正案，而国家继续崩溃，那就是他的责任。如果他不签，崩溃就是总统的责任。"

"那我们怎么办？"新议长问。

沉默。房间里的空气像凝固了一样。七个人，加起来有超过两百年的政治经验，但在这场前所未有的危机面前，他们都像迷路的孩子。

最后，是奥巴马开口了。

"我们需要一个局外人。"他说。

所有人都看向他。奥巴马的声音依然是那种冷静的、像大学教授一样的语调——即使在这种时候。

"现在的问题不是政策。现在的问题是信任。民众不信任白宫，不信任国会，不信任华尔街，不信任媒体。他们需要一个——"他斟酌了一下用词，"一个他们觉得'不是华盛顿的人'的人。"

"你是说特朗普？"丁原的声音里有一丝难以置信。

"我不是说特朗普。"奥巴马说，但他的眼神闪烁了一下，"我是说，我们需要一个能和街上那些愤怒的人说话的人。一个他们觉得'他懂我'的人。而现在——"他停顿了一下，"——不管我们喜不喜欢，特朗普是唯一一个那些人会听的人。"

"他是个灾难，"麦康奈尔说，"你知道他上次执政的时候——"

"我知道他上次执政是什么样的。"奥巴马的声音里第一次有了一丝锐利，"我和他斗了四年。但你看看现在的局面：街上在暴动，货币在崩溃，州政府在独立行动，一半的国民警卫队可能拒绝执行联邦命令。如果我们不做些什么，这个国家会陷入内战。"

"那你有什么建议？"丁原问。

"招安他。"奥巴马说。

这个词在房间里悬了很久。招安。像中国古代的皇帝招安造反的土匪一样。给他一个位置，给他一些权力，让他帮你平定叛乱——然后等叛乱平了，再慢慢收拾他。

"怎么招安？"丁原问。

"2028年大选，"奥巴马说，"我们让他成为共和党候选人。给他政党资源、竞选资金、党机器的支持。但是——"他伸出一根手指，"——我们要选他的副总统。一个我们的人。一个能控制住他的人。"

"彭斯？"丁原立刻想到了那个名字。

"彭斯。"奥巴马点头，"一个虔诚的基督徒，一个可靠的保守派，一个不会被特朗普带偏的人。而且——"他看向麦康奈尔，"——彭斯在国会有人脉。如果特朗普失控了，彭斯是我们的保险栓。"

"你觉得他会同意？"莫雷诺问，"选彭斯当副总统？"

"他会同意的，"丁原说。这是他第一次在会议上展现出那种老牌政客的算计——眼睛眯起来，嘴角微微上扬，像一个看到了机会的棋手。"他需要党的钱。他需要党的组织。上次他赢的时候，他以为是他自己赢的——但其实是RNC的地面游戏帮他赢了密歇根和威斯康星。他这次需要我们。"

"而且，"麦康奈尔补充，声音还是那么慢吞吞的，"我们可以给他政策上的甜头。让他觉得他真的在'抽干沼泽'。给他几个行政命令。让他建墙。让他加关税。那些不重要的东西可以给他。重要的是——"他环视房间，"——他进了白宫之后，我们有国会，我们有最高法院，我们有彭斯。他是总统，但他被包围了。他会像所有其他总统一样，学会和体制合作。"

"如果他不合作呢？"小布什问。

所有人都看向他。小布什的脸上有一种复杂的表情——他见过不合作的总统吗？不，他自己就是总统。他见过的是那种以为自己能改变华盛顿、最后被华盛顿改变的人。

"他会合作的，"丁原说，"他们都合作了。"

这句话在房间里没有引起任何异议。因为在那个房间里的每个人——除了奥巴马，他是个例外——都相信一件事：华盛顿是一台机器。不管你带着多大的决心和怒火走进来，这台机器最终会把你磨成它想要的形状。你可能是一头野牛，但国会是缰绳，法院是围栏，媒体是皮鞭，官僚体系是迷宫。再野的牛，最终也会学会拉犁。

他们都错了。

他们不知道的是，这头"野牛"不是一个人走进华盛顿的。他的身后有一个棋手。一个他们看不见的棋手。那个棋手不在这间会议室里，不在RNC的名单上，不在任何智库或游说公司的工资单上。那个棋手在曼哈顿的五十八层，看着屏幕上的数据，下着他们看不懂的棋。

他们以为他们在招安董卓。他们不知道的是，董卓本来就是要来的——而他们的"招安"，只是给他打开了城门。

**经典场景·海湖庄园的"交易"：**

2027年4月，佛罗里达，海湖庄园。

丁原带着三个人飞抵棕榈滩：RNC的财务主席、RNC的政治总监、以及麦康奈尔的副手（麦康奈尔本人没来——他不想被看到和特朗普握手的照片）。他们在海湖庄园的餐厅吃了午饭——特朗普请客，牛排和可乐，特朗普自己吃了两份冰淇淋。

饭后，特朗普和丁原进了书房。

书房很大，墙上挂满了特朗普和各种名人的合影——和拳王阿里的，和里根的，和杰克逊的，和克林顿夫妇的（那张照片的位置比较偏）。书架上摆着很多书——大部分是特朗普自己写的《交易的艺术》，各种版本各种语言。

"鲍勃，"特朗普坐在巨大的皮椅上，身体往后靠，双手放在肚子上，"你来看我，我很高兴。你带钱来了吗？"

丁原坐在他对面。他的拐杖靠在椅边。他来之前准备了很长的说辞——关于党的团结、关于国家的危机、关于保守派的未来。但特朗普开门见山的方式让他不得不调整策略。

"唐纳德，"丁原说，"党准备全力支持你。"

"真的？"特朗普的眉毛挑了起来，语气里有明显的嘲讽，"上次你们支持的是那个谁——德桑蒂斯？然后是那个黑家伙，斯科特？然后是那个女的，黑利？他们都在哪儿呢，鲍勃？"

"初选还没开始——"

"我领先三十个点。三十个。在每一个摇摆州。"特朗普往前倾了倾身体，声音突然变得尖锐起来，"你们花了四年时间想找人替代我。四年。结果呢？没人。没有人能替代我。因为那些人都是假的。德桑蒂斯是假的，黑利是假的，斯科特是假的。他们在电视上学我说话，但他们不是我。人们看得出来。"

丁原沉默了一会儿。他在心里承认——特朗普说得对。

"我们支持你，唐纳德，"丁原说，"但有条件。"

"什么条件？"

"副总统。"

特朗普的眼睛眯了起来。"我选副总统。"

"你可以选——但要从党推荐的名单里选。"

"名单上有谁？"

"彭斯。"

特朗普做了个鬼脸。"迈克？迈克太无聊了。他每次和我吃饭都要祷告。"

"彭斯能带来福音派的选票。你需要福音派。"

"我已经有福音派了。他们爱我。"

"你有百分之七十的福音派。有了彭斯，你有百分之九十。"丁原的声音很平静，像一个在和客户谈判的老销售——不催促，不施压，只是陈述事实。"而且彭斯在国会有关系。他当了十二年众议员，四年副总统。他知道怎么让法案通过。你想建墙，你需要国会拨款。你想减税，你需要国会投票。彭斯能帮你搞定这些。"

特朗普盯着他看了很久。然后他突然笑了——那种大大的、夸张的、电视上的笑。

"好吧，鲍勃。迈克就迈克。反正副总统也没什么事做。"

丁原松了一口气。他觉得他赢了。他觉得他完成了这次交易——给特朗普政党资源，换取对副总统人选的控制权。他觉得彭斯是套在特朗普脖子上的缰绳。

他不知道的是，就在他和特朗普谈条件的时候，在海湖庄园另一翼的一个房间里，Brother Wang正在和特朗普的竞选经理谈话。谈话的内容是关于" ground game"——不是RNC的地面游戏，而是一个完全独立于RNC的竞选组织。这个组织不向丁原汇报，不向RNC汇报，甚至不向库什纳汇报。它直接向Brother Wang汇报。

丁原以为他控制了钱袋子。但Brother Wang已经准备好了另一个钱袋子——一个比RNC更大、更隐秘、更不受控制的钱袋子。暗钱。超级PAC。社交媒体小额捐款。海外账户。丁原看到的是冰山露出水面的部分。水面之下，是十倍于此的力量。

丁原告辞的时候，特朗普送他到门口。在门廊上，特朗普拍了拍丁原的肩膀——力道很大，拍得这位老人一个趔趄。

"鲍勃，你不会后悔的。我们会赢很多。我们会赢那么多，你都会厌倦赢的。"

丁原微笑着点头。他觉得自己刚刚完成了一笔精明的交易——用一个副总统的位置，换来了对一位不可控候选人的控制权。他不知道，这笔交易的另一方根本没有打算遵守交易的条款。

在回华盛顿的飞机上，丁原的政治总监问他："你觉得他会听话吗？"

丁原看着窗外佛罗里达的海岸线慢慢消失在云层下面，说："他没有选择。"

他错了。

### 2.3 2028大选：MAGA海啸

2028年11月8日，选举日。

结果在当晚十点就确定了——比所有人预期的都快。特朗普赢了44个州。420张选举人票。普选票领先18个百分点——这是自里根1984年连任以来最大的压倒性胜利。

他不仅赢了传统的红州——他赢了密歇根、宾夕法尼亚、威斯康星（2016年的铁锈地带），还赢了明尼苏达、新罕布什尔、内华达、亚利桑那、甚至——让所有民调机构大跌眼镜——弗吉尼亚和科罗拉多。

"这不是胜利，"CNN的评论员在直播中面色苍白地说，"这是海啸。"

在曼哈顿的选举夜集会上——特朗普没有在海湖庄园等结果，而是回到了纽约——特朗普站在舞台上，面前是数以万计穿着MAGA帽子的支持者。红色的帽子像一片海洋。 confetti从天花板上落下来。小唐纳德、埃里克、伊万卡、库什纳站在他身后。蒂芙尼和巴伦站在更远的地方。

彭斯站在特朗普的右手边——按照礼仪应该站的位置。他面带微笑，鼓着掌，但眼神一直在扫视人群。那些人不是在为他欢呼。那些人甚至不知道他是谁。他们只认识一个人。

"谢谢你们！"特朗普对着麦克风大喊。他的声音因为连日的竞选集会已经沙哑了，但那种沙哑反而让他的声音更有力量——像一头嘶吼的狮子。"谢谢你们，我的美国同胞们！"

人群在喊："再干四年！再干四年！再干四年！"

"你们叫我小丑？"特朗普突然说，声音从高亢变成了低沉——那种他在集会上最有杀伤力的语调转变。"你们叫我野蛮人？你们在华盛顿管理了这个国家五十年，结果呢？"

人群安静了一瞬。

"工厂搬走了！工作没了！边境开放了！华尔街把你们的钱都偷走了！五十年！你们管理了五十年！你们有哈佛的学位，你们有华尔街的关系，你们有媒体的话筒——结果呢？"

他停顿了一下。然后举起右手，食指指向天空。

"我至少建了墙！"

人群爆炸了。不是欢呼——是咆哮。一种原始的、从胸腔深处发出的咆哮。那种声音让站在舞台后面的特勤局特工下意识地把手伸向了腰间的武器——不是因为他们觉得人群有危险，而是因为那种声音的规模和烈度，超出了他们训练过的任何场景。

在舞台的侧面阴影里，一个人站在那里。他没有上台。他没有和特朗普的家人站在一起。他甚至没有穿竞选团队的衣服——他穿了一件普通的深色风衣，混在工作人员中间几乎看不见。

Brother Wang。

他看着眼前的人海——那些尖叫的、哭泣的、穿着红色帽子举着标语的普通人。他看着台上的特朗普——那个七十八岁的老人，在聚光灯下像一个获得了整个王国的国王。他看着彭斯——那个站在阴影边缘的副总统，脸上挂着僵硬的微笑，像一个参加别人婚礼的宾客。

他拿出手机，发了一条加密信息。信息只有三个字：

"第一步。"

信息的接收者在世界的另一端——或者说，在华尔街五十八层的办公室里。帕克收到信息后，看着屏幕上的全球市场数据。特朗普胜选的消息已经传遍了全世界。亚洲股市在开盘后暴跌了百分之六。欧洲股市跌了百分之四。但美元在涨——奇怪地涨。因为市场在恐慌，但恐慌中的资本会流向它认为"安全"的地方。而特朗普承诺的"美国第一"——不管政治上意味着什么——对资本来说意味着保护主义、减税、和军事力量。

帕克开始执行Brother Wang三周前给他的指令：在市场恐慌中买入。不是买股票——买基础设施，买能源，买土地，买那些"特朗普行情"会受益的硬资产。用Mnemosyne的钱，用暗钱网络的钱，用那些经过十七层壳公司伪装的、追溯不到任何来源的钱。

在选举夜的狂欢中，没有人注意到这些交易。没有人注意到，在特朗普发表胜选演讲的四十分钟里，有超过两百亿美元的资产悄悄易手。

历史在聚光灯下上演。但真正的革命，发生在阴影里。

**经典场景·竞选总部的战情室：**

选举夜，特朗普竞选总部，曼哈顿特朗普大厦十五层。

这里是真正的战情室——不是电视上看到的那个选举夜派对现场，而是一个没有窗户的大房间，墙上挂满了电子屏幕，显示着各州的实时投票数据。一百多个数据分析员坐在电脑前，电话声此起彼伏。

竞选经理布拉德·帕斯卡尔坐在房间中央的指挥席上。他面前有八部电话——分别通往不同的关键州、法律团队、公关团队、RNC联络人、以及……一部红色的电话。那部红色电话直接连通海湖庄园——不，是连通海湖庄园里某个人。

八点整。第一个州的结果出来了——印第安纳，红。意料之中。

八点三十分。肯塔基，红。西弗吉尼亚，红。

九点。俄亥俄——特朗普领先十二个点。帕斯卡尔松了一口气。俄亥俄是关键。

九点十五分。佛罗里达——特朗普领先十八个点。房间里响起一阵欢呼。佛罗里达是特朗普的主场，但这个领先幅度远超预期。

九点四十五分。北卡罗来纳——特朗普领先九个点。

十点。宾夕法尼亚——特朗普领先七个点。

帕斯卡尔盯着屏幕上的数字。宾夕法尼亚是民主党的铁壁——2020年特朗普输掉了它。但现在，选票统计到百分之六十的时候，他领先七个点。

"布拉德，"一个数据分析员叫他，"你看看密歇根。"

帕斯卡尔看向密歇根的屏幕。特朗普领先十一个点。十一个点。密歇根——民主党铁票仓，2016年特朗普靠不到一万票的差距赢了它，2020年输了。现在他领先十一个点。

"这不对，"帕斯卡尔喃喃自语。

"什么不对？"

"这个数字。我们的内部民调显示密歇根领先三到四个点。但这是十一个点。"

"出口民调呢？"

"出口民调显示——"数据分析员看着屏幕，脸色变了，"出口民调显示拉丁裔男性投特朗普的比例是百分之五十二。五十二。2020年是百分之三十二。"

帕斯卡尔拿起红色电话。响了一声就接通了。

"王，"他说，"你看到密歇根了吗？"

"看到了。"那个平静的声音。

"拉丁裔男性——你做了什么？"

"我告诉过你，"Brother Wang的声音没有任何波动，"社交媒体算法。我们在TikTok和Instagram上针对拉丁裔男性做了六个月的定向推送。不是传统的竞选广告——是内容。拳击视频。创业故事。边境犯罪新闻。'民主党把你们当作理所当然'的叙事。我们没有告诉他们投特朗普。我们只是让他们觉得——"他停顿了一下，"——民主党不在乎他们。"

"这合法吗？"

电话那头沉默了一秒。然后是一声轻笑。

"布拉德，你在选举政治里待了多久了？你真的要在选举夜问我这个问题吗？"

帕斯卡尔没有说话。他看着墙上的屏幕——一个又一个州变红。威斯康星红了。明尼苏达——明尼苏达！自1972年以来从未投过共和党的明尼苏达——红了。

"我们要赢了，"帕斯卡尔说。声音里有一丝他自己都没意识到的颤抖。不是兴奋——是恐惧。因为他开始意识到，这场胜利的规模太大了。大到让他害怕。压倒性的胜利不是授权——是空白支票。而特朗普手里有空白支票的时候，他会写什么？

"我们要赢了，"Brother Wang同意，"但赢只是开始。"

"然后呢？"

"然后，"那个声音说，"才是真正的棋。"

电话挂了。帕斯卡尔看着手里的红色电话，看着墙上满屏的红色——红色，红色，红色，像一片血海。

在房间的角落里，一个年轻的实习生正在用手机拍视频。他想记录这个历史性的夜晚。他拍了欢呼的数据分析员，拍了满屏的红色地图，拍了帕斯卡尔坐在指挥席上接电话的身影。

他没有看到——因为他不知道要看——在帕斯卡尔放下红色电话之后，他的手在发抖。

### 2.4 就职与清洗：入京

2029年1月20日，就职日。

华盛顿。国家广场。

天气很冷——零下二度，风寒指数零下十度。但广场上挤满了人。特勤局估计有两百万人。两百万人站在零下的气温里，等着看那个七十八岁的男人宣誓就职。

特朗普的就职演说持续了四十七分钟。比任何现代总统的就职演说都长。他没有使用提词器——或者说，提词器开着，但他根本不看。

"今天，我们不只是把权力从一个政府移交给另一个政府，"他站在国会山西侧的讲台上，手按在两本圣经上（一本是他母亲给他的，一本是林肯用过的），"我们是在把权力从华盛顿的沼泽里夺回来，交还给你们——美国人民。"

欢呼声。

"很长时间以来，华盛顿的一小群人享受着政府的利益，而人民承担着代价。华盛顿繁荣了——但人民没有分享到那份财富。政客们发达了——但工作离开了，工厂关门了。建制派保护了自己——但他们没有保护你们的工作、你们的边境、你们的自由。"

"他们的胜利不是你们的胜利。他们的成就不是你们的成就。他们在首都庆祝的时候，全国各地辛勤工作的家庭却在挣扎——那些家庭，才是这个国家真正的脊梁。"

他的声音越来越高亢。

"从今天开始，一切都变了！从今天开始，美国优先！每一个决定——不管是贸易、税收、移民、外交——都将为了美国工人和美国家庭的利益而做出！我们将带回我们的工作！我们将重建我们的边境！我们将重建我们的军队！我们将重建我们的国家！"

人群在咆哮。"USA！USA！USA！"

在台上，彭斯站在特朗普身后。他穿着一件厚重的大衣，鼻子冻得通红。他的手按在圣经上的时候，他在祷告——嘴唇无声地翕动。他在求什么？求上帝指引他？求上帝给这个国家智慧？还是求上帝给他自己力量，去面对即将到来的一切？

在台下，丁原坐在贵宾席上。他裹着一条红色的围巾——共和党的颜色。他鼓着掌，脸上带着微笑。但他的眼睛一直在看彭斯。彭斯是他的棋子。彭斯是他的保险栓。只要彭斯在副总统的位置上，特朗普就不能完全失控。

丁原是这么想的。

就职后的第一周，特朗普开始了他的"清洗"。

不是军事清洗——没有坦克上街，没有深夜逮捕。这是21世纪的美国，清洗不需要枪。清洗用笔和电话。

第一天，他签署行政命令，冻结了所有联邦机构的新法规制定——"任何我没有亲自批准的新法规，都不许发布。"

第二天，他解雇了联邦调查局局长——不是因为局长做了什么，而是因为"我需要一个忠诚的人"。

第三天，他解雇了国家情报总监。

第四天，司法部长递交了辞呈——所有人都知道他是被"鼓励"辞职的。

第五天，他签署了"Schedule F"行政命令——这个命令将数以万计的联邦雇员从"公务员保护"中剥离出来，重新分类为"可以随时解雇的政治任命人员"。这意味着他可以清洗整个联邦官僚体系——那个被他称为"深层政府"（Deep State）的体系。

华盛顿震动了。

"他在用行政命令做独裁者！"CNN的头条尖叫。

"国王唐纳德一世。"《纽约时报》的评论版标题。

但支持他的人不在乎。"抽干沼泽！"他们在推特上喊。"终于！"福克斯新闻的主持人在节目上拍桌子。

丁原在RNC总部看着新闻，脸色越来越沉。他打电话给彭斯。

"迈克，你看到那些行政命令了吗？"

"我看到了。"

"你不能让他这么做。"

"我是副总统，鲍勃。我没有权力阻止他签署行政命令。"

"你可以公开反对他。你可以——"

"我可以什么？"彭斯的声音第一次有了一丝锋利。"我可以辞职？然后呢？然后他会任命一个更听话的副总统。我在这里——至少我还在这里。"

丁原挂了电话。他坐在办公室里，看着窗外的国会山。国会山的圆顶在冬日的阳光下闪闪发光——那个他看了四十年的景象，突然变得陌生起来。

他开始意识到，他可能犯了一个错误。

不是招安特朗普这个决定——在当时的情况下，那似乎是唯一的选择。错误在于，他低估了特朗普。他以为特朗普是一头会被制度驯服的野牛。但野牛不会被驯服——野牛只会把围栏撞碎。

而围栏，正在碎裂。

---

**经典场景·椭圆形办公室的第一次内阁会议：**

就职后第二周，白宫。

椭圆形办公室。内阁成员围坐在弧形的沙发和椅子上。副总统彭斯坐在特朗普右手边——那个传统上属于副总统的位置。国务卿、财政部长、国防部长、司法部长（代理，因为正式任命还没通过参议院确认）等坐在两边。

库什纳坐在角落里——他没有正式的内阁职位，但所有人都知道他在那里。伊万卡也在——同样没有正式职位，但同样在那里。

房间的角落里，靠着壁炉，站着一个人。没有座位。没有名牌。他穿着深色西装，没打领带，双手背在身后，像一个安静的侍者。没有人介绍他。没有人问他是谁。只有库什纳偶尔投来一道锐利的目光。

Brother Wang。

特朗普坐在坚毅桌后面——不是在桌子后面的椅子上，而是靠在桌子前面，半坐半站，像一个在更衣室里给队员训话的教练。

"好了，各位，"特朗普说，扫视了一圈房间，"我先说几件事。第一——我们赢了。赢得很大。这意味着人民给了我们授权。很大的授权。不是那种'你们试试看'的授权，是'去他妈的干吧'的授权。"

有人笑了。彭斯没有笑。

"第二——华盛顿的人会告诉你们，有些事不能做。他们会说'这不符合惯例'、'这不符合程序'、'总统先生，我们一直都是这么做的'。你们知道我怎么回答吗？"

他环顾房间。没有人说话。

"我说：'那就是为什么你们这些废物搞了五十年都没搞成。'"

这次没有人笑了。

"第三——忠诚。我不在乎你们是不是最聪明的人。我不在乎你们是不是毕业于哈佛。我在乎的是，你们是不是我的人。如果你们是我的人，我会为你们战斗。我会为你们挡子弹。但如果你们不是——"他停顿了一下，表情从亢奋变成了冰冷，"——我会比你们见过的任何人都狠。"

沉默。空调的嗡嗡声像蚊子叫。

"好了，"特朗普突然又切换回了那种大大咧咧的语气，"开始干活吧。"

内阁成员们开始鱼贯而出。彭斯走在最后——他想和特朗普说几句话，关于那些行政命令的法律风险。但当他走到门口的时候，特朗普叫住了他。

"迈克，等一下。"

彭斯转过身。

特朗普走到他面前，比他高出半个头。他凑近了一些——近到彭斯能闻到他身上的古龙水和汉堡的气味。

"迈克，我知道你和丁原通过电话。"

彭斯的血液凝固了。

"我——"

"别紧张。"特朗普拍了拍他的肩膀，力道很重。"丁原是个好人。他是党的人。但是迈克——"他凑近彭斯的耳朵，声音低沉，"——你是我的副总统。不是党的副总统。我的。你明白吗？"

彭斯看着特朗普的眼睛。那双眼睛——小的、锐利的、在橘黄色的皮肤下像两颗钢珠——里面没有愤怒，没有威胁，只有一种近乎天真的期待。他真的在等彭斯说"我明白"。他真的认为这是一个正常的上级对下属的要求。

"我明白，总统先生。"彭斯说。

"好。"特朗普又拍了拍他的肩膀，这次更重。"好。去干活吧。"

彭斯走出椭圆形办公室。走廊里，他靠在墙上，闭上眼睛。他的心脏在胸腔里狂跳。他做了二十多年的政治。他见过各种类型的政治家——腐败的、高尚的、聪明的、愚蠢的。但他从未见过这样的人。一个把"忠诚"说得如此赤裸裸、如此天真、如此——危险的人。

在他身后，椭圆形办公室的门关上了。房间里只剩下特朗普、库什纳、和那个一直站在角落里的人。

"你觉得他靠得住吗？"特朗普问，朝门的方向歪了歪头。

"现在靠得住，"Brother Wang说，"但'现在'是关键词。"

"什么意思？"

"意思是，当压力足够大的时候，每个人都会做出选择。我们需要确保，当彭斯做出选择的时候，他选择的是对我们有利的方向——或者，让他根本没有选择的机会。"

"怎么做？"

Brother Wang没有回答。他走到窗边，看着南草坪。特勤局的狙击手在屋顶上巡逻。远处，华盛顿纪念碑在灰色的天空下像一根刺向天空的手指。

"董先生，"他说，"你下过围棋吗？"

"没下过。看起来无聊。"

"围棋里有一个概念，叫'气'。每颗棋子都有气——周围的空位就是它的气。气没了，棋子就死了。要杀死一颗棋子，不是直接去吃它——而是堵住它所有的气。"

"所以？"

"彭斯现在有气。丁原是他的气。国会里的保守派是他的气。福音派教会是他的气。那些'深层政府'的官僚是他的气。要控制他，我们不需要威胁他——我们只需要堵住他的气。一条一条地堵。等他发现自己没有气的时候，他要么投降，要么——"

"要么什么？"

"要么死在棋盘上。"

特朗普看着Brother Wang的背影。他不太理解围棋的比喻。但他理解"气"——他一辈子都在和"气"打交道。政治中的"气"是人脉，是选项，是退路。当你没有退路的时候，你就只能往前——或者崩溃。

"堵吧。"特朗普说。

Brother Wang转过身，微微点了点头。然后他从口袋里拿出手机，开始发信息。信息的内容是一串数字和字母——加密的指令，发给分布在不同城市的不同的人。

在华盛顿的酒店房间里，一个"记者"收到了一条信息，开始写一篇关于"彭斯副总统可能在2032年挑战特朗普"的"分析文章"——这篇文章会在合适的时候出现在合适的媒体上，让特朗普怀疑彭斯的忠诚。

在纽约的对冲基金办公室里，帕克收到了一条信息，开始缓慢地、隐蔽地做空几家与彭斯关联的超级PAC的捐赠者的股票——不是为了赚钱，是为了发送信号：如果你支持彭斯，你的钱会缩水。

在海湖庄园，一个"社交名媛"收到了一条信息，开始准备她的"佛罗里达假期"——她的名字叫Celeste。她的任务是进入海湖庄园的社交圈，接近副总统。

棋盘上的子，一颗一颗地落下。

彭斯还不知道。丁原还不知道。甚至特朗普也不完全知道——他只知道他的军师在"做事"，他不需要知道细节。

这就是Brother Wang和特朗普的合作方式：特朗普是矛，Brother Wang是罗盘。矛只需要往前刺。罗盘决定方向。

---

### 2.5 1月6日：火烧洛阳

2029年1月6日。国会山。

这一天本来应该是一个仪式性的日子——国会参众两院联席会议，正式认证选举人团投票结果。这是一个走了两百年的程序：副总统作为参议院议长，主持会议；各州选举人票被一一宣读；结果被正式确认；新总统——等等。

不。特朗普已经在1月20日就职了。

这个1月6日不是认证选举结果。这个1月6日是——

让我们倒回去。

就职之后，特朗普签署了一系列行政命令，但其中有三项引发了最大的争议：

第一项，宣布南部边境进入"国家紧急状态"，授权动用军事力量驱逐无证移民。

第二项，对中国、墨西哥、加拿大和欧盟全面加征百分之二十五的关税。

第三项，也是最具争议的一项：签署行政命令，要求司法部对"2020年和2024年选举舞弊"进行"全面调查"，并授权调查委员会"传唤任何人员、调取任何文件"。

第三项命令在全国引发了轩然大波。因为"任何人员"包括前总统、前副总统、前内阁成员、国会议员——以及，最让两党建制派惊恐的是，最高法的大法官。

"他在用行政命令搞政治迫害！"众议院议长约翰逊在记者会上声音颤抖。

1月3日，众议院议长约翰逊宣布，众议院将对特朗普的"滥用权力"展开正式弹劾调查。

1月4日，参议院多数党领袖麦康奈尔私下告诉共和党参议员，他"不会排除"投票定罪。

1月5日，特朗普在推特上发文："明天来华盛顿。会很疯狂。#停止窃国"

这就是为什么1月6日有数百万人聚集在国家广场。

---

清晨六点。天还没亮。

第一批MAGA支持者已经开始在国家广场聚集。他们来自全国各地——有的坐大巴，有的开车，有的搭便车，有的甚至步行了几百英里。他们穿着红色的MAGA帽子，举着"停止窃国"、"特朗普2029"、"抽干沼泽"的标语。有退伍军人，有卡车司机，有农民，有工厂工人，有带着孩子的家庭——还有一些不那么和平的人：穿着战术装备的"骄傲男孩"、"百分之三者"、以及各种民兵组织的成员。

气温在零度以下。人们的哈气在冷空气中像白烟。但没有人离开。

上午十一点。特朗普出现在椭圆广场——白宫和华盛顿纪念碑之间的椭圆草坪。他站在一个临时搭建的舞台上，面前是无边无际的人群。

"我们今天聚集在这里，是因为我们的国家正在被偷窃！"他对着麦克风大喊。声音通过扩音器传到广场的每一个角落，传到国家广场，传到国会山的方向。"他们想偷走你们的选票！他们想偷走你们的自由！他们想偷走你们的国家！"

"fight！fight！fight！"人群在喊。

"你们永远不要用软弱夺回你们的国家！"特朗普的声音拔高到了嘶哑的极限，"你们要展示力量！你们要走到国会山去——我会和你们一起去！——你们要让那些懦夫听到你们的声音！"

他没有和他们一起去。他回到了白宫，坐在椭圆形办公室里看电视。

但人群去了。

---

下午一点。国会山。

国会山的警察准备不足。他们预计有几万人——不是几百万人。而且他们中的许多人，私下里是支持特朗普的。当人群涌过来的时候，一些警察拉开了围栏。另一些警察后退了。

不是所有的——一些警察坚守岗位，和人群搏斗。但人数差距太大了。几十人对几百万人——不，冲击国会山的不是几百万人，是几百万人中的几千个最激进的。但那几千人就像一把锥子，在防线上撕开了一个口子。

下午两点。第一批抗议者冲破了国会山的警戒线。

下午两点十五分。参众两院的联席会议被紧急中断。议员们被疏散——一些人通过地下通道，一些人被护送到安全的房间。彭斯被特勤局从参议院议长席上拉走——他当时正在主持会议，还在试图维持秩序。

"我不离开，"他对特勤局特工说，"这是我的职责——"

"先生，我们必须走。现在。"特工几乎是把他架走的。

下午两点三十分。抗议者涌入了国会大厦的圆形大厅。有人爬上了雕像，有人坐在议长的椅子上拍照，有人拿走了议员的文件和笔记本电脑——

然后有人放了火。

谁放的火？后来的调查永远无法确定。有人说是一个来自西弗吉尼亚的煤矿工人。有人说是一个来自加州的反法西斯分子——安提法混进来栽赃。有人说是联邦调查局的煽动者。也有人——极少有人——注意到，在火焰燃起之前的几分钟，一个穿着普通黑色外套、戴着棒球帽的身影在国会大厦的一个侧厅里停留了四十秒，然后消失在了人群中。那个人的动作太快了，监控录像只拍到了一个模糊的背影。

但火是真的。

火焰从国会图书馆的侧厅开始——那里收藏着超过一百万册书籍和手稿，包括杰斐逊总统的私人藏书。然后蔓延到了走廊。然后蔓延到了圆形大厅。木制的桌椅、地毯、挂毯、历史文件——一切可燃的东西都在燃烧。

烟柱从国会山的圆顶上升起——那个在美国首都天际线上矗立了两百年的白色圆顶，现在冒出了浓密的黑烟，像一根黑色的柱子刺向灰色的天空。

---

下午三点。白宫屋顶。

Brother Wang站在白宫的屋顶上。特勤局特工试图阻止他——但他出示了一张证件，特工看了一眼就让开了。那张证件不是白宫工作人员证。没有人知道那是什么证件。

他看着国会山方向的烟。黑烟在冬日的天空中格外醒目，像一道伤疤。他戴着无框眼镜，镜片反射着火光——或者说，反射着远方国会山的方向，火在那里燃烧，但从白宫屋顶看不到火焰，只能看到烟。

伊万卡找到了他。她从楼梯口走出来，裹着一件大衣，脸上是震惊和恐惧的混合表情。

"王先生——总统在找你。他在椭圆形办公室——他在看电视，他——他好像不知道该怎么办。"

Brother Wang没有转身。他继续看着烟。

"王先生？"

"你看到了吗？"他说。声音很轻，像是在自言自语。

"什么？"

"那个烟。"

伊万卡看向国会山方向。黑烟正在变浓。她的手捂住了嘴。

"国会——国会在燃烧。"

"不是国会，"Brother Wang说。他终于转过身来。镜片后面的眼睛在暮色中看起来是黑色的——深不见底。他的声音平静得可怕，像在描述一幅画而不是一场灾难。

"你烧的不是国会，"他说，"是旧美国。"

伊万卡愣住了。她不知道该怎么回应这句话。她甚至不确定这句话是对她说的——还是对他自己说的，或者对某个不在场的人说的。

"总统——"她说。

"告诉总统，"Brother Wang说，"发一条推文。'保持和平。记住，我们是法律与秩序的政党。尊重国会警察。回家吧——我们的战斗会继续，但今天，回家。'"

"就这些？"

"就这些。"

"但是火——"

"火会烧完的。旧的东西烧完了，新的东西才能建起来。"

伊万卡看着他。在那一瞬间，她理解了为什么库什纳害怕这个人。不是因为他邪恶——邪恶是可以对抗的。而是因为他在看着一场灾难，一场可能撕裂整个国家的灾难，而他的眼神里没有恐惧、没有兴奋、没有愤怒——只有一种平静的、近乎审美的欣赏，像一个画家在看自己完成的画作。

她转身下楼。

Brother Wang一个人留在屋顶上。风很冷，把他的大衣吹得猎猎作响。他拿出手机，拍了一张照片——黑烟在灰色天空下的国会山。然后他把手机放回口袋，继续看着。

在他身后，华盛顿纪念碑在暮色中变成了一个黑色的剪影。在他面前，国会山在燃烧。

这就是他的"混沌兵法"的第二课：秩序不是被推翻的——秩序是在所有人的注视下，自己崩塌的。你不需要推倒一堵墙。你只需要让足够多的人不相信那堵墙还能保护他们，然后那堵墙自己就会倒。

今天，两百年的"美国秩序"——那个关于宪法、制度、和平交接权力的神话——在火焰中化为灰烬。

明天，新的东西将从灰烬中诞生。什么东西？那取决于谁来定义"新"。

Brother Wang已经想好了谁来定义。

---

**经典场景·椭圆形办公室里的沉默：**

下午四点。椭圆形办公室。

电视开着——福克斯新闻，画面是国会山的实时航拍。黑色的烟柱清晰可见。主播的声音在颤抖："……国会山在燃烧……重复……国会大厦着火了……目前还不清楚是否有人员伤亡……华盛顿特区已经宣布进入紧急状态……国民警卫队正在部署……"

特朗普坐在坚毅桌后面。他的脸上没有表情——不是冷静，而是一种空白。他的手放在桌上，手指无意识地敲击着桌面。他面前的手机在震动——推特、电话、短信，潮水般涌进来。但他没有看。

库什纳站在窗边，背对着房间，看着窗外的方向——从白宫看不到国会山，但他能看到烟。

彭斯坐在角落里的沙发上。他的脸色惨白。他的圣经放在膝盖上，翻开着，但他没有在读。他的嘴唇在动——祷告。

门开了。Brother Wang走了进来。

"王！"特朗普看到他，像是从恍惚中惊醒，"你看到了吗？他们烧了国会。他们烧了国会！"

"我看到了。"Brother Wang走到电视前，看了一眼画面，然后转过身，"董先生，我们需要谈一下。"

"谈什么？我们需要叫军队进来！我们需要——"

"军队已经在路上了。但在那之前，你需要做一件事。"

"什么事？"

"发一条推文。让你的人回家。"

特朗普瞪大了眼睛。"回家？他们在为我战斗！你让我叫他们回家？"

"如果你不叫他们回家，"Brother Wang的声音依然平静，但平静下面有一丝锐利——那种他极少使用的、直接的、不绕弯子的语气，"如果今天晚上国会山变成一场屠杀，如果军队和你的支持者在国会山的台阶上交火——那你就完了。不是政治上完了——是历史上完了。你会被记住为那个在美国本土引发内战的总统。"

"但是——"

"发那条推文。告诉他们你爱他们。告诉他们要和平。告诉他们回家。然后——"他停顿了一下，"然后，你可以说这一切都是安提法的错。是反法西斯分子混进来放火的。是深层政府策划的。你的人会相信你。他们永远相信你。"

特朗普看着他。然后看着库什纳。库什纳点了点头。然后看着彭斯。彭斯没有说话——他甚至没有抬头。

特朗普拿起手机。他开始打字——用两根手指，很慢，很用力，像在凿石头。

"我要求所有在国会山的人保持和平。不要使用暴力。尊重我们伟大的国会警察。你们是真正的爱国者。回家吧。记住——我们永远在战斗。我们会赢。我爱你们！"

他按下了发送键。

推文在三十秒内被转发了十万次。

在国会山，人们开始——不是立刻，但慢慢地——离开。不是所有人。有些人留下了，被国民警卫队逮捕。但大多数人，看到特朗普的推文后，开始往回走。火焰还在燃烧，但人群在散去。

那天晚上，国会山的火在夜里十点才被完全扑灭。

损失：国会图书馆侧厅完全烧毁——约三十万册书籍和无数历史文件。圆形大厅的壁画被烟熏黑。众议院议事厅的部分天花板坍塌。三人死亡——一个国会警察，两个抗议者。超过一百四十人受伤。

"火烧洛阳"。

在《三国演义》里，董卓火烧洛阳是为了不给联军留下任何东西——焦土政策。在这个故事里，火烧国会山不是特朗普下令的——至少不是直接下令的。但火确实达到了Brother Wang想要的效果：旧的共识在火焰中死去了。在1月6日之后，美国不再是那个"可以和平解决分歧"的美国了。那个美国在烟柱升上天空的时候，就已经死了。

剩下的只有权力——赤裸裸的、没有伪装的、不被制度约束的权力。谁有胆量拿起它，谁就拥有它。

### 2.6 副总统的裂痕：连环计铺垫

1月6日之后，华盛顿变了。

不是制度变了——制度还在，国会还在开会（在隔壁的图书馆大楼里，因为国会大厦需要修复），法院还在开庭。但空气变了。在华盛顿工作了几十年的老政客们说，那种感觉就像"地震之后"——房子还在，但地基裂了，你永远在等着下一次震动。

特朗普和彭斯之间的关系，在1月6日之后出现了一条肉眼可见的裂缝。

裂缝不是在公开场合出现的——在公开场合，彭斯依然是那个忠诚的副总统，站在特朗普身后，点头、微笑、鼓掌。但在私下里，他们几乎不再说话。椭圆形办公室的会议上，彭斯坐在他的位置上，一言不发。特朗普不再问他的意见。特朗普问意见的时候，他问库什纳，他问竞选经理，他打电话给福克斯新闻的主持人——但他不再问彭斯。

彭斯开始更多地待在海军天文台的副总统官邸，而不是白宫西厢。他开始更多地和"老朋友"吃饭——丁原、麦康奈尔、众议院议长约翰逊、福音派领袖。他开始一个人在官邸的花园里长时间地散步，读圣经，祷告。

2029年2月中旬，海湖庄园。

总统日周末。特朗普在海湖庄园招待支持者——这是他就职后第一次回到佛罗里达。海湖庄园变成了一个MAGA的节日现场：高尔夫球场、泳池派对、筹款晚宴、深夜的马球吧聚会。

彭斯也来了。他不想来——但作为副总统，他不能不来。他带着妻子凯伦，在庄园里待了三天。三天里，他和特朗普的交谈时间加起来不到十分钟。

第三天晚上，在海湖庄园的一场私人晚宴上，彭斯遇到了一个人。

她站在露台的角落里，手里端着一杯白葡萄酒，穿着一件简单的深蓝色晚礼服，金色的头发松松地挽在脑后。她不是那种在海湖庄园常见的女人——不是整容过度的富婆，不是寻求拍照机会的社交 climber，不是特朗普身边那些涂着浓妆的年轻女孩。她有一种安静的气质——那种在喧闹的派对中自成一个世界的气质。

彭斯不认识她。但他注意到她在看他——不是那种谄媚的、想和副总统合影的看，而是一种温和的、带着好奇和理解的看。

他走了过去。

"副总统先生，"她开口了。声音很轻，带着一点南方口音——路易斯安那？还是密西西比？"我不想打扰您。我只是想说——我为您祈祷。"

彭斯愣了一下。在华盛顿，每个人都想从他这里得到什么——投票、支持、合影、承诺。没有人说"我为您祈祷"。

"谢谢你，"他说，"你是？"

"Celeste，"她说，微笑了——一个温暖但不过分热情的微笑，"Celeste——我是一个朋友的朋友。来自新奥尔良。"

"你在华盛顿工作吗？"

"不，我只是来参加周末的活动。我——"她停顿了一下，像是在斟酌用词，"我是一个信徒。我知道这些日子对您来说很艰难。在信仰的道路上行走，尤其是在权力的走廊里——"

她没有说完这句话。但她不需要说完。彭斯的眼睛里闪过一道光——那种"终于有人理解我"的光。

"你读圣经吗？"彭斯问。

"每天读，"Celeste说，"今天早上我读的是《但以理书》。但以理在狮子坑里——他 surrounded by 敌人，但他信靠上帝。"

"《但以理书》第六章第二十二节，"彭斯立刻接上，"我的神差遣使者，封住狮子的口，叫狮子不伤我。"

"因我在神面前无辜，我在王面前也没有行过亏损的事。"Celeste接了下一句。

两人对视了一秒。在海湖庄园的喧闹声中——音乐声、笑声、杯盏碰撞声、特朗普在某个房间里传来的大笑声——他们站在露台上，在佛罗里达温暖的夜风中，隔着一杯白葡萄酒和一句经文，完成了第一次接触。

她不是偶然出现在这里的。她的"新奥尔良出身"是精心设计的——彭斯在印第安纳的教会支持网络和新奥尔良的福音派教会有联系。她的"但以理书"是精心选择的——但以理是"在异教王宫中持守信仰"的先知，是彭斯这种人最能共情的圣经人物。她的穿着、她的微笑、她说话的节奏、她看他的眼神——全部是被训练过的。

她是小孟——残兵的渗透专家。她是Celeste——海湖庄园的新社交名媛。她是貂蝉——连环计的核心。

但彭斯不知道。在他眼里，她只是一个在喧闹的派对里唯一一个"理解"他的人。一个和他一起读圣经的人。一个不会向他要求任何东西——至少看起来不会——的人。

在露台的另一端，库什纳看到了这一幕。他皱了皱眉——彭斯和一个陌生女人说话。他拿起手机，准备让安保团队查一下这个女人的身份。

"不用查。"

一个声音在他身后。库什纳转身——Brother Wang站在那里，手里端着一杯苏打水（他不喝酒）。

"你认识她？"库什纳问。

"她是朋友。"Brother Wang说。

"什么朋友？"

"一个在彭斯需要朋友的时候出现的朋友。"

库什纳盯着他。"你在做什么？"

"我在给他一条路。"Brother Wang看着露台那头的彭斯和Celeste——他们已经换了一个话题，正在聊什么，彭斯的脸上第一次露出了这个周末以来第一个真正的微笑。"一个人觉得自己没有路的时候，是最危险的。我要让他觉得他有路。"

"通向哪里的路？"

Brother Wang喝了一口苏打水。"通向他以为是上帝指引的方向。"

库什纳没有再问。他学会了——和Brother Wang相处，你不能问太多问题。不是因为他不回答，而是因为你可能不喜欢答案。

但在心里，库什纳记下了这个女人的脸。他会让自己的人——不是特朗普的人，不是RNC的人，是他自己的人——去查她。

这是Brother Wang没有预料到的——或者说，他预料到了，但他不在乎。在他的计算中，库什纳的疑心是一个变量，但不是一个致命的变量。库什纳现在不会行动——因为他需要Brother Wang。而等到库什纳决定行动的时候，棋盘上的局势已经不可逆了。

连环计的第一颗子，已经落下。

### 2.7 丁原之死：RNC主席的政治终结

2029年3月。华盛顿。

丁原在RNC主席的位置上坐了十二年。十二年里，他见过总统上台、总统下台、政党轮替、丑闻爆发、危机降临——他都挺过来了。他以为他能挺过特朗普。

他错了。

特朗普对丁原的打击不是正面的——没有公开谴责，没有推特攻击，没有解雇。特朗普不需要正面攻击。特朗普做的事情更狠：他抽走了丁原的权力基础。

第一步，钱。特朗普的竞选团队建立了一个独立于RNC的筹款平台——"拯救美国行动委员会"（Save America Action）。这个平台直接向特朗普的支持者募集捐款，不需要经过RNC。从1月到3月，"拯救美国"募集了三亿七千万美元——而RNC同期只募集了四千二百万。没有钱，党主席就没有权力。没有钱，你就不能支持候选人，不能投放广告，不能组织地面游戏，不能回报忠诚的盟友。

第二步，人。特朗普通过他在各州的支持者，开始替换各州的共和党党部成员。一个一个州，一个一个县，忠于丁原的"老共和党人"被特朗普的支持者在党部选举中投票罢免。到3月底，五十个州的共和党委员会中，有三十一个已经被MAGA派控制。

第三步，规则。RNC的规则是党主席制定的——但党主席是由RNC委员选举产生的。当大部分委员变成了特朗普的人，丁原的主席位置就保不住了。

3月18日，RNC春季会议。棕榈滩——就在海湖庄园几英里之外。

丁原走进会议室的时候，他就知道完了。会议室里的168名RNC委员——每个州三名，包括州党部主席和两名全国委员——有超过一百人不看他的眼睛。他们看着自己的手机，看着天花板，看着地板——就是不看他。

会议开始。第一项议程就是选举新的RNC主席。

丁原发表了演讲。他的声音依然是那个沉稳的、老练的、带着四十年政治重量的声音。他谈到了共和党的历史——林肯的党、里根的党、自由企业和强大国防的党。他谈到了团结——"我们是一个党，不是一个人的党"。他谈到了未来——"如果我们变成了个人崇拜，我们就不再是政党了"。

他演讲的时候，有人在听。有些人甚至在点头。但鼓掌的人很少。

然后是投票。

第一轮：丁原，37票。特朗普支持的候选人——一个来自佛罗里达的MAGA众议员，名叫迈克尔·"迈克"·沃尔茨——112票。其他候选人分了剩下的19票。

37票。四十年的政治生涯，五任总统，无数次竞选胜利，最终换来37票。

丁原在结果宣布后站起来。他拄着拐杖走到讲台上——他本来不需要拐杖，他已经用了三个月，但今天他觉得他需要它。他看着台下的168张脸——其中有很多是他亲手提拔起来的人，是他在他们还是县党部志愿者的时候就认识的人。那些人现在不看他。

"我祝贺沃尔茨议员，"丁原说。他的声音很平稳——四十年的政治训练让他在最糟糕的时刻也能维持表面的体面。"我会尽全力配合权力交接。共和党大于任何个人——包括我。"

散会后，他一个人走出会议室。佛罗里达的阳光刺眼。他站在酒店门口的台阶上，看着停车场里的汽车——大部分车上贴着特朗普的标语。他的司机把车开过来，他上了车。

"去哪里，丁先生？"司机问。

丁原想了想。"机场。回家。"

在回华盛顿的飞机上——他坐的是商业航班，RNC的私人飞机已经不归他调遣了——他接到了彭斯的电话。

"鲍勃，我很抱歉。"

"没什么可抱歉的，迈克。这就是政治。"

"你要怎么办？"

丁原看着舷窗外的云层。他七十二岁了。他的妻子在2023年去世了。他的孩子都在私营部门工作，不想碰政治。他在华盛顿有一栋房子，在俄亥俄有一个农场。他可以退休——写回忆录、打高尔夫、在大学里讲课。这是大多数政客退场后的生活。

但他不是大多数政客。他在华盛顿的漩涡中心待了四十年。四十年里，他见过太多人被打败——但那些人都是被对手打败的。他不一样。他是被他自己"招安"进来的人打败的。他亲手打开了城门。

"迈克，"丁原说，"听我说。"

"我在。"

"那个人——他不是一个正常的总统。你知道的。"

"鲍勃，我——"

"让我说完。你是副总统。宪法赋予你的权力不多，但有一项权力——"他停顿了一下，"——如果他变得……如果他变得无法履行职责……你知道第二十五修正案怎么说。"

电话那头沉默了很久。

"鲍勃，我不能——"

"我不是让你现在做任何事。我是让你记住——当那个时刻到来的时候，你有选择的权利。而且不只是权利——你有责任。你在上帝面前宣誓过。"

他挂了电话。

飞机在三万五千英尺的高空向东飞行。丁原闭上眼睛。他想起了2016年——第一次特朗普竞选的时候，他在RNC总部的办公室里看着电视上的初选结果，当时他对幕僚说："这个小丑不可能赢。"

他想起了2027年——在那个橡木护墙板的会议室里，奥巴马说"我们需要一个局外人"，他觉得这是个精明的交易。

他想起了2028年——在海湖庄园的书房里，特朗普拍着他的肩膀说"鲍勃，你不会后悔的"。

他后悔了。

但丁原的"死"不是一个老人的退场。丁原虽然失去了RNC主席的位置，但他没有消失。他回到了华盛顿，回到了他在乔治城的房子里，开始做一件他四十年都没有时间做的事：他开始组织。秘密地。和麦康奈尔、和约翰逊、和民主党领袖、和那些"老共和党人"——那些和他一样被特朗普扫地出门的人。

他不再是RNC主席了。他没有钱，没有职位，没有官方权力。但他有一样东西特朗普拿不走——他知道体制的每一个裂缝、每一条暗道、每一个可以被撬动的支点。

他知道杀死一个总统不需要子弹。你只需要在正确的时间、正确的地点，让正确的人说出正确的话。

他开始策划。不是政变——至少他自己不认为是政变。他认为自己是在"拯救共和党"和"拯救国家"。历史上每一个密谋者都这样认为。

在乔治城的书房里，他打开了一本《三国演义》——这是他的亚裔幕僚多年前送给他的，他之前从没翻过。他翻到了"王允巧使连环计"那一回。

他读了一整夜。

天亮的时候，他合上书，看着窗外初升的太阳。他拿起电话，拨了一个号码——不是打给麦康奈尔，不是打给彭斯。是打给一个他二十年没联系过的人：共和党党鞭王允。

"老王，"他说，"我需要和你谈谈。关于……关于怎么拯救这个国家。"

电话那头沉默了三秒。然后王允说："你来我这里。别带手机。"

丁原不知道的是——他的每一通电话、每一次会面、每一步行动，都在Brother Wang的视线之内。Brother Wang没有阻止他。Brother Wang甚至在某些时候——通过极其隐蔽的方式——"帮"了他：让他更容易接触到某些人，让他的信息更流畅通，让他觉得自己的密谋正在顺利进行。

因为Brother Wang需要丁原来做一件事：把王允推到台前。丁原是明线——一个公开的、已经失势的对手，可以被特朗普随时碾压。王允是暗线——那个真正危险的人，藏在丁原身后。

等丁原的利用价值结束的时候，他就会像一颗被吃掉的棋子一样，从棋盘上被拿掉。

但那是连环计成熟之后的事了。

### 2.8 郿坞：海湖庄园

2029年4月。棕榈滩，佛罗里达。

1月6日之后，特朗普在华盛顿待的时间越来越少。他讨厌白宫——白宫太小了，太阴郁了，太多人看着他。他喜欢海湖庄园。海湖庄园是他的。他在1985年用不到一千万美元买下了这个庄园（当时它是一个破败的私人住宅，曾经是玛乔丽·梅里韦瑟·波斯特的豪宅，波斯特去世后把它捐给了联邦政府作为总统度假胜地，但联邦政府维护不起，又卖回给了私人市场——特朗普买了下来）。他把它改造成了一个私人俱乐部——会员费二十万美元入门，每年一万四千美元会费。

对特朗普来说，海湖庄园不止是一个家。它是一个堡垒，一个宫殿，一个避难所。在海湖庄园，他被朋友包围着——会员、捐助人、忠实追随者。在海湖庄园，他不需要面对华盛顿的敌意，不需要面对媒体的追问，不需要面对那些"深层政府"官僚的阴奉阳违。在海湖庄园，他可以打高尔夫、吃牛排、看福克斯新闻、在深夜打电话——一切都像他当总统之前的样子。

他开始把越来越多的总统事务搬到海湖庄园来处理。周末一定回佛罗里达。周三下午就走，周一上午才回华盛顿。外国元首来访问——请到海湖庄园来。内阁会议——在海湖庄园开。情报简报——让情报人员飞到佛罗里达来做。

华盛顿的政治评论员们开始称海湖庄园为"冬季白宫"——但它不止是冬季。它是特朗普的"郿坞"。

在《三国演义》里，董卓在长安以西二百五十里的地方修建了郿坞——一个巨大的城堡，囤积了足够三十年用的粮食、黄金、白银、珠宝、丝绸。他说："事成，雄踞天下；不成，守此足以毕老。"——如果我赢了天下，郿坞无所谓；如果我输了，守着郿坞也能过一辈子。

特朗普的海湖庄园就是他的郿坞。

这里有他的私人保安——不是特勤局（虽然特勤局也在），而是他自己雇佣的安保团队，由前纽约警察和前特勤局特工组成，直接向他汇报。这里有他自己的通讯系统——加密电话、卫星互联网、不经过白宫通讯局的线路。这里有他的人——俱乐部的会员里有对冲基金经理、房地产开发商、媒体老板、外国游说者、军火商。这些人是他真正的"朝廷"——不是内阁里那些部长，而是在马球吧里和他喝酒的那些亿万富翁。

4月的一个周末，海湖庄园举办了一场特别的活动：不是筹款晚宴，不是高尔夫球赛，而是一场"战略会议"。参加的人只有不到二十个：特朗普、库什纳、小唐纳德、埃里克、伊万卡、竞选经理帕斯卡尔、新任RNC主席沃尔茨、特朗普的私人律师朱利安尼（已经八十多岁了，但还在）、福克斯新闻的几个主持人、两个最大的超级PAC捐助人——以及Brother Wang。

会议在海湖庄园的"丘吉尔厅"举行——一个镶着深色木护墙板的房间，墙上挂着温斯顿·丘吉尔的肖像（特朗普收藏了很多丘吉尔的纪念品，他喜欢把自己和丘吉尔比较）。

特朗普坐在主位——一张巨大的桃花心木桌子的尽头。他面前放着一杯健怡可乐，旁边是一叠打印出来的推特评论和福克斯新闻的新闻稿。

"好了，各位，"特朗普说，"我们现在的情况怎么样？"

帕斯卡尔翻开他的笔记本。"民调支持率——百分之五十二。考虑到1月6号的事，这已经很高了。基础选民支持率——百分之九十四。铁锈地带——我们的支持率比选举日还高了两个点。"

"国会呢？"

"众议院——约翰逊的议长位置不稳。我们的人正在逼他下台。如果我们能换掉他，众议院就是我们的了。参议院——"帕斯卡尔看了一眼库什纳，"参议院更复杂。麦康奈尔还在。"

"麦康奈尔，"特朗普的语气变得阴沉，"那个老乌龟。我给了他减税。我给了他最高法院大法官。他怎么对我的？"

"麦康奈尔不是最大的问题，"库什纳开口了。他说话的时候，所有人都看向他——因为他很少在这种会议上先开口。"最大的问题在法院。"

"法院？我们有六个保守派大法官——"

"最高法院不是问题。问题是下级法院。全国有超过八百个联邦法官职位——我们上任以来只任命了四十七个。剩下的都是奥巴马、克林顿、甚至布什任命的。他们在全国范围内签发禁令，阻止我们的行政命令执行。"

"那就换掉他们！"特朗普说，"Schedule F——我签了那个命令，我可以解雇联邦雇员——"

"法官不是联邦雇员，总统先生。法官是终身任职的。除非弹劾，你不能撤换他们。"

"那就弹劾他们！"

"弹劾需要众议院通过、参议院三分之二定罪。我们在参议院没有三分之二。"

特朗普一拳砸在桌子上。健怡可乐的杯子跳了一下。

"那我做这些有什么用？我签行政命令，法官给我停了。我要建墙，国会不给钱。我要清理深层政府，他们说我是独裁者。我赢了史上最大的选举胜利——结果呢？什么都没变！"

房间里沉默了几秒钟。然后Brother Wang说话了。

他一直坐在桌子的最末端——那个最不引人注目的位置。他面前没有文件，没有电脑，只有一杯绿茶。整个会议他一句话都没说。但当他开口的时候，所有人都安静了——因为他们都知道，当这个中国人开口的时候，说的话一定很重要。

"董先生，"他说，声音很轻，慢得像在给一个孩子解释一道数学题，"你在试图用总统的权力来解决一个问题。但这个问题不是总统能解决的。"

"什么意思？"

"你面对的不是一个政策问题。不是一个法律问题。你面对的是一个系统性问题。这个系统——国会、法院、官僚机构、媒体——被设计来阻止变化。它已经运行了两百年。它有自己的免疫系统。任何试图改变它的人，都会被免疫系统攻击。"

"那你说怎么办？"

Brother Wang站起来。他走到房间里挂着的一幅美国地图前面——不是选举地图，是一幅老式的、标注着军事基地和联邦设施分布的地图。

"你不需要控制华盛顿。"他说。

"什么？"

"华盛顿是一个舞台。所有的政治表演都在华盛顿上演。但真正的权力不在华盛顿。真正的权力在——"他的手指在地图上移动，"——在各州。在国民警卫队。在能源网络。在金融系统。在媒体平台。在那些华盛顿的人看不起的地方。"

他转过身，看着房间里的人。

"董卓进了洛阳之后，他犯了一个错误——他留在了洛阳。他以为控制了首都就控制了天下。但天下不在首都。天下在地方。等到诸侯联合起来讨伐他的时候，他发现洛阳是一个陷阱——四面受敌，无险可守。所以他烧了洛阳，迁都长安。然后在长安外面建了郿坞。"

"你在说什么？"小唐纳德问。他是特朗普几个孩子里最像父亲的——冲动、好斗、但不太懂战略。

"我在说，"Brother Wang的目光移回到特朗普身上，"董先生，你需要在华盛顿之外建立你的权力基础。不要把所有的精力都放在和国会打架上。把精力放在控制州政府上。把精力放在国民警卫队上——给他们更多的钱、更多的装备、更多的授权。把精力放在能源上——控制了石油和天然气，你就控制了经济的命脉。把精力放在媒体上——不是福克斯，福克斯已经是你的了。我说是社交媒体。把精力放在金融上——华尔街现在怕你，但怕不等于忠诚。让他们赚钱，他们就忠诚。"

他走回桌子，坐下来。

"华盛顿是一个围城。如果你所有的兵力都在围城里，城外的敌人把你围死只是时间问题。你需要在城外有军队——真正的军队。"

房间里安静了很久。特朗普看着Brother Wang，然后看着库什纳。库什纳微微点头——他不完全同意Brother Wang的方案，但他同意方向：华盛顿的僵局无法在华盛顿内部打破。

"好，"特朗普最终说，"我们这么干。贾里德——你负责金融和中东那边。小唐——你去和各州的党部谈。埃里克——你管竞选筹款。沃尔茨——你管RNC的地面游戏。"

他看向Brother Wang。

"王——你负责……"他挥了挥手，"你负责你一直在做的事。"

Brother Wang微微点头。

会议结束后，人们三三两两地离开。库什纳走在最后——他等所有人都走了之后，走到Brother Wang身边。

"郿坞。"他说。

"嗯？"

"你讲的那个郿坞的故事。你说董卓建了郿坞，以为能守一辈子。"

"对。"

"但董卓最后还是死了。"库什纳看着他，"郿坞没有救他。"

Brother Wang端起绿茶，喝了一口。

"郿坞没有救董卓，"他承认，"因为董卓犯了另一个错误。"

"什么错误？"

"他以为郿坞是堡垒。但堡垒最大的危险不是外面的敌人——是里面的人。"他看着库什纳，镜片后面的眼睛没有任何情绪，"董卓不是死在城外的敌人手里。他死在他最信任的人手里。他的义子。吕布。"

库什纳的背脊发凉。

"你是在说彭斯？"

"我是在说，"Brother Wang放下茶杯，"任何堡垒都防不住内鬼。在我们的堡垒里——"他停顿了一下，"——我们需要确保没有内鬼。或者，如果有内鬼，我们知道他是谁，并且让他在最关键的时候做出对我们有利的选择。"

"那个女人——Celeste。"

"她在做她的工作。"

"如果她失败了？"

Brother Wang微微一笑。那是库什纳最不喜欢的微笑——那种"你以为你看到了我的计划，但你只看到了第一层"的微笑。

"她不会失败。但如果她失败了——"他站起来，拍了拍库什纳的肩膀（这是一个极其罕见的亲昵动作），"——我们还有Plan B。还有Plan C。还有Plan D。一盘棋不是一颗子。连环计也不止一环。"

他走了。库什纳一个人留在丘吉尔厅里，看着墙上的丘吉尔肖像。丘吉尔在二战中说过："我们抗争到底，我们在法国作战，我们在海上作战，我们在天空作战——我们绝不投降。"

但丘吉尔赢了。而董卓输了。

库什纳想知道——在Brother Wang的棋局里，他们是丘吉尔，还是董卓？

### 2.9 诸侯觉醒：两党建制派+媒体+科技巨头+司法部联盟

2029年5月。丹佛。

丹佛"宪法恢复会议"——在共通文档中，这个会议发生在2027年中，由加州州长发起，31个州的代表参加。但在懂王分支中，时间线被推迟了。因为特朗普不是军事入侵——他是通过选举上台的，在就职初期仍然有大量民众支持，"抵抗"看起来像是"不承认选举结果"，政治代价太高。

1月6日改变了一切。

国会山燃烧的画面传遍了全世界。在欧洲，盟友们公开质疑美国的民主制度是否还能运转。在中国，外交部发言人用一句"望美国人民早日恢复秩序"的外交辞令表达了不加掩饰的嘲讽。在华尔街，道琼斯指数在1月7日暴跌两千点——虽然在特朗普发推文让支持者回家后反弹了一部分，但资本的信心已经受到了不可逆转的伤害。

但最关键的变化发生在国内。

1月6日之前，反对特朗普的力量是分散的：民主党人恨他，但他们是少数党；"老共和党人"不喜欢他，但他们不敢公开反对，因为害怕初选；媒体骂他，但他们的骂声反而让他的基础选民更坚定；科技巨头不喜欢他，但他们需要流量；司法部……司法部在观望。

1月6日之后，这些力量开始聚集。不是因为他们突然变得勇敢——而是因为他们突然意识到，如果不联合起来，他们都会被碾碎。

5月15日，丹佛。

会议不是在丹佛市中心的会展中心举行的——那样太引人注目。会议在丹佛郊外落基山脚下的一个私人度假村举行。度假村的主人是一位硅谷风险投资家——一位在奥巴马时期担任过驻法大使的科技亿万富翁。度假村的安保由一家私人军事公司负责——这家公司的CEO是前三角洲部队指挥官，和曹操（宾州国民警卫队的那位中校）有旧交。

这一次，参加会议的不只是31个州的代表。

加州州长还是盟主——和共通框架一致。但他身边多了几个在火凤分支中不会出现的面孔：

——一位来自纽约的联邦检察官（代表司法部的"抵抗派"）。

——一位谷歌副总裁（代表科技巨头——他们受够了特朗普的社交媒体威胁）。

——CNN的总裁（代表主流媒体——他们已经成了特朗普的公开敌人）。

——索罗斯的一个代表（代表自由派大金主）。

——以及，最让人意外的，科赫兄弟网络的一位代表（代表保守派大金主——他们也受够了特朗普的关税和不可预测性）。

还有一个没有出现在官方名单上的人——一位28岁的对冲基金经理，来自曼哈顿。他不是任何州的代表，不是任何公司的高管，不是任何政党的官员。他是作为加州州长的"私人朋友"被邀请来的。

司马懿。

会议持续了三天。三天里，与会者们争论、争吵、讨价还价。红州的州长和蓝州的州长互相指责——"都是你们的人纵容他""是你们的人先搞身份政治"。科技巨头想要数据隐私保护的承诺，能源巨头想要解除环保管制，华尔街想要稳定——但稳定是所有人都同意的唯一共同点。

最后一天，加州州长站在度假村的大露台上，面对着落基山脉的落日，发表了一篇演讲。

"我们今天站在这里，不是作为民主党人或共和党人，不是作为自由派或保守派，"他说，声音不大，但通过扩音器传到了每个角落，"我们站在这里，作为美国人。我们的国父们在两百年前建立了一个制度——一个不完美的制度，一个充满妥协的制度，但一个有史以来最伟大的制度。那个制度现在面临着前所未有的威胁。不是来自外国军队，不是来自恐怖分子——而是来自一个人。一个人以为他比制度更大。一个人以为他的意志就是法律。"

"我们在这里，不是为了推翻一个总统——总统是人民选的。我们在这里，是为了捍卫宪法。如果宪法死了，美国就死了。如果美国死了，我们所有人——不管你是红还是蓝——都失去了我们的国家。"

掌声。不热烈——这些人不是特朗普集会上的信徒，他们是精于算计的政客和商人——但掌声是真诚的。

在露台的角落里，司马懿没有鼓掌。他端着一杯威士忌，看着远处落基山脉的雪顶在夕阳下变成金红色。他的嘴角有一丝很淡的笑意——像一个棋手看到对手落下了他预期中的那步棋。

曹操站在他旁边。曹操是以"宾州国民警卫队代表"的身份来的——他不是州长，只是一个中校营长，但他在过去几个月里展现出的军事和组织才能让他成为了东北地区不可忽视的力量。

"你好像不太激动。"曹操说。他也没鼓掌。

"激动什么？"司马懿的声音很轻，带着那种永远的、淡淡的嘲讽。

"他们在组建联盟。他们要讨伐董卓。这不是你想看到的吗？"

"这是我想看到的。"司马懿承认，"但这不是结局。这只是开始。"

"什么意思？"

"孟德，"——他叫曹操的字，这是只有极少数人才会用的称呼——"你觉得联盟里的这些人，有几个是真正为了宪法？有几个是为了自己的利益？加州来的那位想当总统。纽约的检察官想当州长。科赫的人想要减税。谷歌的人想要不被反垄断。索罗斯的人想要……谁知道索罗斯想要什么。他们现在因为共同的敌人站在一起。敌人消失之后呢？"

曹操看着他。"你是说——"

"我是说，董卓是一个问题。但董卓不是最大的问题。"司马懿喝了一口威士忌，"最大的问题是——董卓之后呢？这个国家已经碎了。碎了的东西拼不回来。不管谁赢了董卓，接下来都是群雄逐鹿。而现在站在这个露台上的人——"他环顾四周，目光扫过那些高谈阔论的州长、亿万富翁、将军们，"——他们中的大部分，都会被吃掉。"

"包括你？"

司马懿微微一笑。"我不吃人，也不会被人吃。"

"那你做什么？"

"我看棋。"他把空酒杯放在旁边的栏杆上，"偶尔下一子。"

曹操看着这个比他年轻十岁的男人。他第一次见到司马懿是在曼哈顿的一次私人晚宴上——那个时候他只觉得这是一个说话轻声轻气的华尔街青年，聪明但过于阴柔。现在他不这么想了。在这个年轻人的眼睛里，他看到了一种让他后背发凉的东西——不是野心，不是残忍，而是一种宏大的、超越了个人荣辱的计算。这个人在算的不是下一季度的收益，不是下一次选举的胜负——他在算的是整个国家未来十年、二十年、三十年的走向。

"你和那个中国人是一类人。"曹操突然说。

司马懿的微笑没有变。但他的眼睛——那双永远半眯着的、笑意不达眼底的眼睛——微微收缩了一下。

"哪个中国人？"

"特朗普身边那个。Brother Wang。王博士。不管他叫什么。"

沉默。风从落基山上吹下来，带着松针和雪的气息。

"我知道他，"司马懿最终说，声音比之前更轻了，"我知道他很久了。"

"他是你的人？"

"他不是任何人的人。"

"那他是谁的人？"

司马懿转过头，看着曹操。在夕阳的余晖中，他的脸一半是金色，一半是阴影。

"他是下棋的人。和我一样。"

"两个棋手？同一盘棋？"

"同一盘棋。不同的下法。"司马懿说，"他相信混沌——让一切崩溃，然后在废墟上建立新秩序。我相信——"他停顿了一下，"——我相信信息。谁控制了信息，谁就控制了现实。"

"谁会赢？"

司马懿笑了。这次是真正的笑——不是那种嘲讽的笑，而是一种遇到了同类时的、棋逢对手的笑。

"孟德，你觉得一盘棋里，赢的是黑棋还是白棋？"

"赢的是——"曹操想了想，"——下得更好的那个。"

"不。"司马懿摇头，"赢的是那个制定规则的人。"

他转身走进了度假村的建筑里。曹操留在露台上，看着他的背影消失在门后。远处，落基山脉的最后一缕阳光消失在山脊后面，天色迅速暗了下来。

曹操突然理解了一件事：这场战争——不管它以什么形式到来——不只是特朗普和反特朗普势力之间的战争。它是两个棋手之间的战争。Brother Wang和司马懿。混沌和信息。一个在东海岸，一个在华尔街。一个在特朗普的影子里，一个在所有人的影子里。

而他——曹操——和所有那些在露台上高谈阔论的州长、将军、亿万富翁一样，都只是他们棋盘上的棋子。

这个认知让他感到一阵寒意。但紧接着——让他自己都感到意外的——是一种兴奋。一种战士在听到战鼓声时的兴奋。

如果他们以为他只是一颗棋子——那他们会大吃一惊的。

### 3.2 分支关键差异点总结

1. **入京方式（核心差异）**：选举而非军事。特朗普通过合法选举上台，这给了他"民主授权"的合法性——反对他的人不能简单地说他是"入侵者"，而必须在制度框架内对抗他。这也让1月6日的"火烧洛阳"更具悲剧性——不是外敌焚烧首都，而是国家内部的撕裂导致了自我伤害。

2. **丁原之死**：政治消灭而非肉体消灭。丁原不是被吕布杀死的——他是被自己的政党抛弃的。但这种"死法"更符合政治惊悚的调性，也更贴近现实：在现代政治中，权力的丧失不需要刀剑，只需要选票和钱。

3. **吕布的"背叛"动机**：在原著和火凤分支中，吕布背叛董卓是因为貂蝉（美色）+ 野心 + 对丁原的旧情。在懂王分支中，彭斯的"背叛"是因为良心和信仰——他真的相信他在捍卫宪法。这让他成为一个更悲剧的人物，而不是一个单纯的"三姓家奴"。

4. **时间线整体后移**：共通框架中"董卓入京"在2027年初。懂王分支中特朗普2028年11月当选、2029年1月就职，"火烧洛阳"在2029年1月6日，诸侯会盟在2029年5月。整体后移约1-2年，因为选举需要时间。

5. **十常侍的定义**：共通框架中十常侍是"白宫影子内阁"（幕僚长+国安顾问+情报总监三人组）。懂王分支中，十常侍扩展为整个"华府建制派/Deep State"——包括官僚体系、游说集团、媒体精英、科技巨头、华尔街。三驾马车只是这个体系的冰山一角。

6. **小孟的渗透目标**：从"王允身边"变为"海湖庄园社交圈"。小孟需要同时接近特朗普和彭斯——在特朗普面前，她是一个能给彭斯"枕边风"的社交名媛；在彭斯面前，她是一个能理解他信仰的"属灵伙伴"。双重身份的难度更大，戏剧张力更强。

## 第四章 懂王版暗线备忘

> **作者专用**：记录懂王分支中已埋设和计划埋设的暗线，供后续写作参考。标注[已埋设]的暗线已在前文叙事中出现，标注[计划中]的暗线为后续章节预留。

### 4.1 Brother Wang暗线

1. **[已埋设] Mnemosyne的统一暗记**：Brother Wang和司马懿都在使用Mnemosyne（记忆女神）这个名字——对冲基金是Mnemosyne Capital，基金会是Mnemosyne Foundation。这暗示了两个棋手之间的深层联系：他们可能知道彼此的存在，甚至可能有某种历史渊源。但目前没有人（包括特朗普和库什纳）知道这个联系。

2. **[已埋设] Brother Wang的证件**：1月6日白宫屋顶场景中，Brother Wang向特勤局出示了一张"不是白宫工作人员证"的证件，特勤局看了一眼就让开了。这张证件是什么？可能是情报机构的最高级别通行证，可能是军方的某种授权，可能是他自己伪造的——但伪造到能骗过特勤局的程度意味着他在情报系统内部有人。

3. **[已埋设] 国会山纵火的"四十秒身影"**：在火焰燃起之前几分钟，一个穿黑色外套戴棒球帽的身影在侧厅停留了四十秒。这个身影是谁？是Brother Wang的人？是残兵的人？是司马懿的人？还是——最让人不安的可能——是多个势力同时派了人，而火是多方合力的结果？作者暂不揭示。

4. **[已埋设] Brother Wang的围棋隐喻**：他多次用围棋来解释策略——"气"、"废子"、"中腹落子"。他的棋风是"中国流"——吴清源开创的布局。这暗示他可能和吴清源有某种个人联系（师徒？家族？），或者他自视为"改变棋道的人"。

5. **[计划中] Brother Wang的真实身份**：他的MIT记录显示他2005年毕业，2002年之前"不存在"。2002年之前他在哪里？中国？他是否和中国政府有联系？还是说他和司马懿一样，是某个更古老的"棋手家族"的成员？这个暗线将在董卓篇之后才开始回收。

6. **[计划中] Brother Wang和司马懿的关系**：曹操说"你和那个中国人是一类人"时，司马懿没有否认。这暗示他们可能曾经认识——可能在MIT，可能在华尔街，可能在更早的时候。他们是师兄弟？是同一个组织培养出来的？还是在某个看不见的"棋手圈子"里的对手？

7. **[已埋设] Brother Wang的银色戒指**：第一章描写他左手无名指戴一枚极其普通的银色戒指，不是婚戒。这枚戒指的来历将在后续揭示——可能是某个组织的标记，可能是他和某个重要人物的信物。

### 4.2 特朗普暗线

1. **[已埋设] 深夜电话**：特朗普在深夜打电话给"不在官方顾问名单上的人"——其中最重要的是Brother Wang，但Brother Wang不是唯一一个。他还打给福克斯主持人、脱口秀电台主持人、各州党主席等。这些深夜电话是特朗普真正的决策渠道——白宫幕僚们看到的只是冰山一角。

2. **[计划中] 特朗普对Brother Wang的真实态度**：特朗普信任Brother Wang吗？他依赖他的建议——但特朗普一辈子信任过任何人吗？他是否也在防备Brother Wang？就像他防备所有人一样？这条暗线将在特朗普和库什纳的私下对话中逐步揭示。

3. **[已埋设] 特朗普的"直觉"**：Brother Wang让库什纳"告诉他这是他的直觉，他会相信的"。这说明Brother Wang理解特朗普的核心心理需求——他需要相信所有好主意都是他自己想出来的。Brother Wang在提建议的时候，永远把它包装成"你本来就是这么想的，我只是帮你说出来"。

### 4.3 彭斯/Celeste暗线

1. **[已埋设] Celeste的"但以理书"伪装**：小孟选择用《但以理书》来和彭斯建立联系——"但以理在狮子坑里"的故事对彭斯这种"在异教王宫中持守信仰"的人有极强的共鸣。这不是偶然——小孟（或Brother Wang/司马懿）研究过彭斯的信仰背景，知道哪句经文能击中他。

2. **[计划中] 彭斯的妻子凯伦**：彭斯的妻子凯伦是一个极其重要的角色——她是彭斯"彭斯规则"（不在没有妻子在场时和其他女性单独相处）的来源。Celeste如何绕过凯伦？是让凯伦也信任她？还是利用彭斯和凯伦之间的某些裂痕？凯伦会成为连环计的障碍还是助力？

3. **[计划中] Celeste的双重身份**：她需要同时接近特朗普——特朗普身边有无数年轻漂亮的女人，接近他不难，但要获得他的信任很难。她将以什么身份进入特朗普的视野？是作为彭斯的"朋友"？作为海湖庄园的会员？作为捐助人？还是通过伊万卡或库什纳？

4. **[计划中] Celeste的真实感情**：小孟对彭斯产生了真实的感情吗？在共通框架中，小孟和王蕊有真实的恋爱关系。对彭斯呢？如果她对彭斯产生了真情，连环计就会变得更加复杂——她能完成任务吗？还是会在关键时刻犹豫？

5. **[已埋设] 库什纳在调查Celeste**：库什纳让自己的人去查Celeste的身份。这是一个潜在的炸弹——如果库什纳查到了小孟的真实身份，连环计就会被识破。Brother Wang知道库什纳在查吗？他肯定知道——他知道一切。那他为什么不阻止？因为库什纳查到真相的时间点，也是他计划中的一部分？

### 4.4 库什纳暗线

1. **[已埋设] 库什纳的"两手准备"**：在和伊万卡的对话中，库什纳明确表示要"做两手准备"——如果一切顺利就继续，如果出问题就确保自己安全。这意味着他在为自己留后路。他的后路在哪里？中东——他在沙特和阿联酋有深厚的关系网和个人投资。

2. **[已埋设] 库什纳和Brother Wang的冷战**：两人之间形成了微妙的平衡——库什纳管明线，Brother Wang管暗线；库什纳知道Brother Wang在做他不知道的事，但他没有证据也没有替代方案。这种平衡能维持多久？当平衡被打破时会发生什么？

3. **[计划中] 库什纳和丁原/王允的关系**：库什纳是否知道丁原和王允的密谋？他是否在两边下注——既做特朗普的女婿，又和反特朗普势力暗中联系？这将决定他在最终崩溃时站在哪一边。

### 4.5 丁原/王允暗线

1. **[已埋设] 丁原读《三国演义》**：丁原在政治失势后读了一整夜《三国演义》"连环计"一回，然后主动联系王允。这意味着丁原在策划中借鉴了三国的策略——但他读的是《三国演义》（小说），不是真正的历史。小说中的连环计成功了——但真实历史中，王允的计谋虽然杀了董卓，却也导致了李傕郭汜之乱，王允自己也被杀了。丁原不知道他正在重蹈一个悲剧性的覆辙。

2. **[计划中] 王允的真实计划**：共通框架中王允是共和党党鞭，表面合作暗策划政变。但他的连环计到底是什么？只是利用小孟离间特朗普和彭斯？还是有更深的布局——比如利用第二十五修正案、弹劾、甚至某种宪法危机？王允和司马懿是什么关系？（残兵是司马懿资助的，小孟是残兵的人——所以王允的连环计中，小孟是"借"来的武器。但王允知道小孟背后是司马懿吗？）

3. **[计划中] 王蕊的位置**：共通框架中王蕊是王允的独生女，和小孟（Celeste）有恋爱关系。在懂王分支中，王蕊是否知道Celeste在利用她接近彭斯？她的反应会是什么？当她发现她的恋人不仅是一个"卧底"，而且同时在"勾引"副总统时，她会崩溃？愤怒？还是会成为另一个变量？

### 4.6 诸侯暗线

1. **[已埋设] 司马懿的"名单"**：在丹佛会议上，司马懿不像其他人那样高谈阔论——他在观察。他在制作一份"名单"——哪些人可以利用，哪些人必须消灭，哪些人可以在战后收编。这份名单将决定三国分汉时的格局。

2. **[已埋设] 曹操的野心**：曹操在丹佛露台上意识到自己是棋子——但他决定不当棋子。他将利用联盟的资源发展自己的势力，最终成为北方的霸主。他和司马懿之间将形成一种复杂的合作-对抗关系。

3. **[计划中] 科技巨头/媒体的角色**：谷歌副总裁、CNN总裁出席了丹佛会议。他们在联盟中扮演什么角色？是盟友？还是工具人？Brother Wang对媒体和算法的理解不比任何人差——他会如何反击？社交媒体平台是否会成为新的战场？

### 4.7 共通暗线在本分支的呼应

1. **修车铺老板（燎原火/赵云）**：在共通文档开头，火哥在宾州哈里斯堡开修车铺，深夜练拳，看北方天空。在懂王分支中，这条暗线保持一致——燎原火潜伏在曹操腹地（宾州），等待时机。他和曹操的相遇、加入刘备阵营的故事线在两个分支中大致相同。

2. **写辞职信幕僚之死**：共通文档中那个写了"州牧制度"辞职信后"自杀"的幕僚——在懂王分支中，他的死是Brother Wang暗线网络清理证人的行动，和火凤分支中莫拉莱斯阵营的清理对应。

3. **Mnemosyne做空自由币**：共通文档中Mnemosyne Capital做空自由币获利三十亿——在懂王分支中，这是Brother Wang操作的（而在火凤分支中可能是另一回事？）。等等——共通文档中Mnemosyne是司马懿的基金。这里出现了一个需要解决的问题：如果Mnemosyne是司马懿的，为什么Brother Wang在用它？

   **解答**：Mnemosyne Capital是司马懿的基金，但Brother Wang曾经（可能现在仍然）在其中有某种角色——或者他们之间有某种协议/合作/对抗。更准确地说，在懂王分支中，Brother Wang可能通过自己的渠道（他自己的基金、壳公司网络）进行操作，而Mnemosyne的三十亿是司马懿的独立操作——两个棋手在同一场危机中各自布局，有时互相借力，有时互相干扰。这种"两个棋手在同一张棋盘上下子"的设定增加了叙事的深度。

4. **"吕布是品牌"**：共通文档中丁原把"吕布"打造成一个特种部队品牌。在懂王分支中，丁原的身份变成了RNC主席，"吕布"的设定变为副总统彭斯——"品牌"的暗线需要调整。在懂王分支中，"彭斯"这个品牌不是军事品牌，而是政治品牌——"可靠的保守派""虔诚的基督徒""党可以信任的人"。彭斯一辈子都在扮演这个品牌，直到他必须决定自己是品牌还是人。

## 第五章 军师兵法哲学

> 本章补充懂王分支独有的兵法哲学。共通文档中已定义司马懿（信息兵法）、曹操（速度兵法）、刘备（人心兵法）。懂王分支增加董卓阵营的两位核心军师——Brother Wang（混沌兵法）和特朗普本人（交易的艺术兵法），以及库什纳（交易撮合兵法）作为补充。

### 5.1 Brother Wang：混沌兵法

**核心命题**：秩序是弱者的幻觉。真正的力量存在于混沌之中。

| 维度 | 内容 |
|---|---|
| 哲学根源 | 道家 + 量子不确定性 + 复杂系统理论 + 博弈论。"道生一，一生二，二生三，三生万物"——万物从混沌中诞生。市场不是有效的，政治不是理性的，历史不是线性的。所有试图维持秩序的努力最终都会被混沌吞噬——而智者在混沌中找到机会 |
| 核心信条 | "最好的计划是让所有人以为一切都是意外。" |
| 兵法要旨 | 1. **不制造事件，创造条件**：不直接出手杀人，而是创造一个人必死的环境——市场恐慌、社会撕裂、政治危机，然后让"自然力量"完成剩下的工作。2. **利用情绪作为武器**：愤怒是最强大的政治力量——比军队强大，比金钱强大，比逻辑强大。3. **让敌人自己打败自己**：不需要进攻敌人的堡垒，只需要让堡垒的地基自己裂开。4. **永远留后手**：Plan B、C、D、E——在围棋里这叫"留余味"，一颗棋子在当下看起来没用，但在三十手之后可能成为杀棋的关键。5. **废子利用**：最危险的棋子不是最强的棋子，而是所有人都以为已经死了的废子 |
| 和司马懿信息兵法的区别 | 司马懿相信控制信息就能控制现实——他的策略是监控、操纵、预测。Brother Wang相信信息永远不可能完全控制——混沌系统对初始条件极其敏感，蝴蝶效应不可预测。所以他的策略不是控制，而是**扰动**：在关键节点投入微小的力量，让系统自己走向他想要的方向。这是"控制论"和"混沌理论"的区别 |
| 围棋哲学 | "国际象棋是杀死国王。围棋是比谁占的地盘大。但在最高层次上，围棋不是占地盘——是制造一种局面，让对手发现不管他怎么下，他都已经输了。赢棋的最高境界是让对手自己认输。" |
| 弱点 | 混沌兵法的最大弱点是——混沌不可控。你可以创造条件、扰动系统、引导方向，但混沌系统的演化永远有意外。一个Brother Wang没有预料到的变量（比如库什纳的疑心、小孟的真情、曹操的野心、司马懿的反制），可能让整个计划偏离轨道。Brother Wang知道这一点——这就是为什么他永远留后手。但永远有多远？ |

**Brother Wang的三层布局法则：**

- **第一层：明线**——所有人都看得见的：特朗普的行政命令、MAGA集会、国会斗争、推特攻击。这是给媒体和公众看的。
- **第二层：暗线**——少数人看得见的：暗钱网络、算法操控、媒体叙事、政治勒索。这是给知情者看的。
- **第三层：深渊**——只有Brother Wang自己看得见的：三层壳公司的资金流向、加密通讯网络中的指令、他在每个人身边安插的"眼"（围棋术语，棋子）。这一层不存在于任何文件、任何通讯、任何证人的记忆里——只存在于他的脑子里。

### 5.2 唐纳德·特朗普：交易的艺术兵法

**核心命题**：一切都是交易。所有的关系、联盟、冲突、忠诚、背叛——本质上都是交易。

| 维度 | 内容 |
|---|---|
| 哲学根源 | 尼采"权力意志" + 房地产开发商的实用主义 + 真人秀表演理论 + 个人直觉。特朗普不读书（他声称自己读过《交易的艺术》，但那本书是代写的），他的"哲学"完全来自他七十年人生经验的直觉总结 |
| 核心信条 | "你赢了，就是对的。" "人们会相信你说的话——只要你说得足够大声、足够多次、足够自信。" |
| 兵法要旨 | 1. **永远进攻**：防守是失败者的策略。不管你处于什么位置，永远主动出击——发推特、开集会、签行政命令、解雇人、制造争议。进攻让对手疲于应对，没有时间组织反击。2. **制造混乱然后谈判**：先把桌子掀了，让所有人都惊慌失措，然后你作为"唯一能解决问题的人"来提出条件。这是他在房地产谈判中屡试不爽的招数——先起诉对方，让对方花几百万律师费，然后提出和解。3. **忠诚是单向的**：你必须对我忠诚，但我对你的忠诚取决于你对我有多少用。4. **永不道歉**：道歉是软弱。即使你错了，也要说成是对的——或者说成是别人的错。基础选民会因为你不道歉而更加爱你。5. **拥抱争议**：所有的宣传都是好宣传。哪怕媒体在骂你，他们也在给你免费的曝光时间。没有比"被所有人谈论"更好的政治位置。6. **直觉优于分析**：不要听专家的——他们大部分时候是错的。相信你的gut（直觉）。你的gut能闻到那些书呆子闻不到的东西——人群的情绪、对手的弱点、时机的成熟 |
| 和Brother Wang的关系 | Brother Wang是他缺失的那一半。特朗普有直觉、有魅力、有动员群众的能力——但他没有战略耐心，没有系统思维，没有长期布局的能力。Brother Wang有战略、有计算、有耐心——但他没有魅力，不能站在台上对着百万人演讲。两人的合作是互补的：特朗普是矛，Brother Wang是罗盘。但这种合作有一个根本张力——特朗普永远在即兴发挥，Brother Wang永远在计算。即兴发挥可能打乱计算。计算可能限制即兴。这种张力将是董卓阵营内爆的重要原因 |
| 弱点 | 1. **无法承认错误**：这让他无法从失败中学习。2. **短期思维**：他关注的是今天的头条、今晚的收视率、明天的推特——不是明年、后年、十年后的格局。3. **容易被奉承操纵**：说他想听的话的人，比说真话的人更容易获得他的信任。Brother Wang利用这一点——他永远把建议包装成"这本来就是你的想法"。4. **家族政治**：他信任家人——伊万卡、库什纳、小唐纳德、埃里克——超过任何人。但家族成员各有各的利益和弱点，这会成为裂缝。5. **对制度的无知**：他不理解政府是怎么运作的——他以为当总统就像当CEO，可以随便命令人。制度的反弹力比他想象的大得多 |

**特朗普的谈判法则（来自《交易的艺术》但被实战深化）：**

1. 高开价——比你想要的多要很多，让谈判空间更大。
2. 制造紧迫感——"现在不做就永远没机会了"。
3. 展示实力——让对方看到你的支持者、你的钱、你的媒体影响力。
4. 不可预测——让对手永远猜不到你下一步要做什么。
5. 随时准备翻脸——今天的朋友是明天的敌人，今天的敌人是明天的朋友。
6. 永远给自己留退路——海湖庄园。

### 5.3 贾里德·库什纳：交易撮合兵法

**核心命题**：政治是可能性的艺术——在所有不可能中找到可能的交易。

| 维度 | 内容 |
|---|---|
| 哲学根源 | 哈佛MBA思维 + 房地产家族的务实 + 犹太民族千年生存智慧。库什纳不是一个有意识形态的人——他是一个解决问题的人。他的世界观不是"什么是对的"而是"什么是可行的" |
| 核心信条 | "不喜欢的人也可以握手。不想做的交易也可以做。只要结果是对的。" |
| 兵法要旨 | 1. **永远保持对话**：即使是你的敌人，也要保持沟通渠道畅通。你不知道什么时候需要和他们做交易。2. **多线操作**：不要把所有鸡蛋放在一个篮子里——和特朗普合作的同时，和华尔街保持关系；在美国政坛运作的同时，在中东建立自己的网络。3. **隐身**：真正的权力不在聚光灯下。让特朗普去上头条，让Brother Wang去暗线操作，你只需要在关键节点出现，撮合交易，然后消失。4. **耐心**：交易需要时间。有些交易需要几个月，有些需要几年。不要急于求成。5. **永远计算退出成本**：在进入任何局面之前，先想清楚怎么退出 |
| 在阵营中的角色 | 他是"现实主义的声音"——当特朗普想要冲动行事时，当Brother Wang想要极端操作时，库什纳是那个问"这对我们有什么实际好处"的人。他的声音不一定总是被听到，但他的存在本身就是一种平衡 |
| 弱点 | 他缺乏Brother Wang的远见和特朗普的直觉。他是一个优秀的战术家，但不是一个战略家。他擅长撮合交易，但不擅长创造新的格局。在历史的大转折中，交易撮合者往往被有真正信仰（或真正野心）的人碾压——因为信仰和野心能驱使人们做理性不会做的事 |

### 5.4 兵法哲学的碰撞

董卓阵营的兵法不是单一的——它是三种兵法的混合体，而这三种兵法之间的张力本身就是戏剧冲突的来源：

- **特朗普的交易兵法**是即兴的、进攻的、个人魅力驱动的。它像一把大锤——砸碎一切障碍，但也可能砸碎自己人。
- **Brother Wang的混沌兵法**是系统的、耐心的、长期布局的。它像水——无孔不入，绕过障碍，慢慢侵蚀一切。
- **库什纳的撮合兵法**是务实的、平衡的、风险控制的。它像润滑剂——让机器运转，减少摩擦，但不能驱动机器。

这三种兵法在"正常时期"可以互补——大锤砸碎障碍，水侵蚀根基，润滑剂减少摩擦。但在危机时刻，它们会互相冲突：特朗普想要发推特进攻，Brother Wang想要耐心等待时机成熟，库什纳想要妥协谈判。当这三种声音在椭圆形办公室里同时响起时，特朗普会听谁的？

答案是：他听那个在正确的时间说正确的话的人。而Brother Wang最擅长的就是在正确的时间说正确的话——把他的建议包装成特朗普自己的直觉。

但Brother Wang不能控制一切。当突发事件发生时——比如1月6日的火比预期烧得大，比如彭斯的良心比预期更坚定，比如小孟对彭斯产生了真情，比如库什纳的调查挖到了不该挖的东西——即兴发挥就会取代计算。

而在即兴发挥的战场上，特朗普的大锤可能砸向任何方向——包括砸向自己人。

## 第六章 阅读指引

### 6.1 如何阅读本设定集

1. **先读共通文档**：《世界观设定-共通.md》包含了世界观的基础框架、共通角色设定、核心隐喻和叙事方法论。不读共通文档直接读本文件，会缺失很多背景信息。

2. **再读本文件**：本文件是懂王分支的独有内容，定义了董卓阵营的角色（特朗普、Brother Wang、库什纳/李儒）、分支叙事线、时间线差异、暗线和兵法哲学。

3. **对比阅读（可选）**：如果阅读过火凤分支（《世界观设定-火凤.md》），可以对比两个分支在"董卓入京""火烧洛阳""吕布背叛"等核心事件上的不同处理方式。两个分支共享共通框架和外围角色（曹操、刘备、孙坚、司马懿、残兵等），但董卓阵营和核心事件完全不同。

### 6.2 本分支的叙事特色

**政治惊悚而非军事史诗**：火凤分支是军事+金融的宏大战争叙事，有坦克、有军队、有武装入侵。懂王分支是政治惊悚——核心战场是选举、媒体、社交媒体算法、国会投票、行政命令、最高法院裁决、政党内部清洗。武器不是子弹，是推文、民调、捐款、爆料、程序性投票。

**现实映射但不换皮**：特朗普就是特朗普——不是"一个像特朗普的虚构人物"，而是特朗普本人（在虚构的2028-2029时间线中）。他用他真实的说话方式、真实的政治风格、真实的性格缺陷和优点。其他现实人物（彭斯、库什纳、麦康奈尔等）也是如此。但故事不是"如果特朗普再次当选会怎样"的简单推演——它是在三国框架下的重新叙事，有Brother Wang这样的虚构角色推动情节走向戏剧性的方向。

**限知视角的严格执行**：读者不会看到上帝视角的解释。当Brother Wang在下棋时，读者和特朗普一样不知道他在做什么——只能通过结果反推。当小孟在执行任务时，读者和彭斯一样不知道她的真实身份。暗线不标注、不解释、不点破——最好的暗线是第二遍阅读时才发现"原来这里有伏笔"。

**暗线不影响首次阅读**：虽然有大量暗线，但首次阅读时即使完全没有注意到暗线，故事本身也是完整和通顺的。暗线是给第二遍、第三遍阅读准备的彩蛋。

### 6.3 核心主题

1. **制度 vs 个人**：两百年的美国制度能否约束一个不按规则出牌的人？制度的"免疫系统"在面对真正的威胁时是否有效？还是说制度本身已经腐朽到一推就倒？

2. **秩序与混沌**：Brother Wang相信"混沌中取利"，司马懿相信"控制信息即控制现实"，特朗普相信"直觉和力量"，彭斯相信"信仰和宪法"——不同世界观的碰撞是整个故事的思想核心。

3. **忠诚与背叛**：在极端压力下，什么是忠诚？对个人的忠诚？对政党的忠诚？对宪法的忠诚？对信仰的忠诚？彭斯的悲剧在于他的忠诚对象之间不可调和。

4. **民众的力量与危险**：特朗普的权力来自民众——数百万愤怒的、被剥夺的、渴望改变的民众。民众的力量可以把一个"小丑"送上总统宝座，也可以烧毁国会山。民众是救世主还是毁灭者？故事不给出简单答案。

5. **棋手与棋子**：在历史的大棋局中，谁是棋手？谁是棋子？棋手真的能控制棋局吗？还是说棋局本身有自己的意志，连棋手也被它裹挟？

### 6.4 后续章节提示（董卓篇后半段）

董卓篇将在以下节点继续展开：

- **连环计深化**：Celeste（小孟）同时渗透特朗普和彭斯的信任圈，制造两人之间的裂痕。王蕊发现Celeste的"双重生活"，成为不可预测的变量。

- **丁原-王允密谋线**：失势的丁原和党鞭王允联手，利用丁原对体制的了解和王允的阴谋才能，策划一场宪法框架内的"政变"——弹劾+第二十五修正案+彭斯的倒戈。

- **库什纳的选择**：库什纳对Celeste的调查逐步深入，他发现的东西可能改变一切。他会选择揭露真相？还是选择利用这个信息为自己谋利？还是——最危险的可能——他选择站在错误的一边。

- **诸侯对峙**：丹佛联盟开始对特朗普阵营施加实质压力——法律挑战、州层面的抵抗、经济压力、甚至军事准备。特朗普的应对方式（Brother Wang建议的"郿坞战略"）将导致美国进一步分裂。

- **Brother Wang vs 司马懿**：两个棋手的暗战逐步浮出水面。他们不是盟友，也不是简单的敌人——他们是同一盘棋中的对手，各自下着自己的棋，有时借力，有时拆台。他们的第一次直接对弈，将决定董卓篇的结局。

- **高潮：彭斯的选择**：在最关键的时刻——可能是弹劾投票、可能是第二十五修正案、可能是某个更极端的危机——彭斯必须做出选择。他选择特朗普？还是选择宪法？他的选择将触发连锁反应，最终导致董卓阵营的崩溃。

- **尾声：董卓之死**：不是肉体死亡——在21世纪的美国，前总统不会被斩首（不会在公开场合）。特朗普的"死"是政治死亡——被弹劾、被定罪、或被迫辞职。但他的倒台不会带来和平——只会打开群雄逐鹿的潘多拉魔盒。
